### 完整转录稿

Hermes Agent，也就是 AI 代理人。最近我不管是日常的工作管理，包括写稿的辅助创作，包括这个频道的配图的自动生成流程，甚至包括整个公司的所有意义、无方面的内容、事务性的内容——总之就是跟生产力方面相关的所有东西，我都是交给这些 AI 助手去做了。

先给大家浅浅地解释一下，Hermes Agent 到底是一个什么东西。它跟我们平常一般人用的软件，比如说 ChatGPT 的体验，或者说豆包这种软件不太一样。它不是一个集成好的像聊天形式那样的软件，就是说你说一句它回合制的，说完就截止了。它会做什么呢？它是会在你说了一句话之后去理解你的意图，然后主动根据自己有的技能，把你把事情给做好做完，不需要你一步一步去指示，而是直接给你一个结果——就这样一个助手。

比如说简单举个例子，我让它给我这个脚本配图，它就会读我的脚本，然后去找哪些地方要配图，然后去网上搜资料，结合我的文本信息，要么去截图，要么自己通过 HTML 来生成图片，然后它会自己再打开 HTML 去截图，再把截图放在我文章合适的地方。这样一整套流程，只要前期花时间配置优化好了，我就写脚本，然后丢给它，它框框一路干到底，就帮我把所有配图工作都完成了。

而现在你在跟它对话、让它处理事情的过程中，它是有记忆的。这个记忆是怎么回事呢？它是有对你的记忆，它知道你是一个什么样的人，它知道我喜欢什么、不喜欢什么，它知道我喜欢按照什么样的方式去办事情。更重要的是，它是一个会自动进化的 Agent。也就是说当你让它去做一个事情，它发现它没有做过，那么它就会去网上查怎么做，或者它自己思考一下这个事情要怎么做。做的时候它会踩坑，它会遇到没法完成的事情，然后这时候你可以跟它对话，它会给你反馈，你可以给它指引，它也可能自己去探索。那一整套流程下来，等到这个事情做完之后，它就会把整个事情总结成一个叫做 Skill 的文件，也就是我们刚刚说的那个技能文件。那么下次我们要做类似的事情的时候，它就会先去搜一下自己有没有这个技能，然后看一下原来要做这些步骤，于是它就可以不踩坑地按照以前做好的那个手册，把整个事情更快更好地给你办好。

所以总结下来简单来说，它不是一个聊天机器，它是一个有能力用各种工具、有各种记忆和进化的能力，能通过聊天把你的意图分解成一个一个具体的步骤，然后给每一个步骤的反馈，将下一个步骤推进下去，直到一件事情被完成了——这样一个助手。

不过这个所谓的代理人，它只是一个壳，就像一个驾驶员。驾驶员你要有车才能开动，那那个车就是我们所谓的大模型，它的真正的智力源泉。那这里的大模型我们选用什么呢？其实可以有很多选择，比如说你可以用 DeepSeek，你可以自己本地部署千问 3.2 72B。如果你有钱的话——你是真的钱多到没地方花的话，感觉我这辈子钱真的花不完了的话——而且你甚至可以用 Claude。那就是简单部署的方式。

一开始我是把它部署在自己的本地主机上，然后用本地的方式去跑一些 7B、6B 的模型，用下来感觉是非常不错的，自己做一些小程式、处理一些日常的电脑事务都完全没有问题。不过后来是因为这个物理舒适度上的原因，我放弃了本地部署。特别是在 DeepSeek V4 更新完之后，从一开始浅尝一下，到后来的重度依赖。再到后来我对移动端也有了工作的需求，比如说我想在手机上，不管在合适合地，不管我电脑有没有开着，我都想用这个 Agent。于是我租了一个云服务器，在上面部署了自己的 Hermes Agent，这也就是我现在用的一个比较完全体了。不管在哪里，不管是用手机还是用电脑，都能通过聊天软件对话的方式，完成我所有的工作。

那么接下来也可以给大家详细讲一下，我用 Hermes 到底是用来干什么的。最后还会提一下，用 Hermes Agent 加 DeepSeek V4 来做这些事情，包括整个服务器，一共要花多少钱——肯定会超出你的预料。

那么接下来我会按整个事情的复杂度来做一个排序，我会从一些非常简单的小功能，到后面越来越复杂的功能，再到最后一个——我跟你说，光是最后一个它能办到的事情，就足以让我每年省下上万块钱和至少十几天的时间。

**小功能一：To Do List**

我现在可以通过手机上的聊天软件，我跟它说我接下来要做什么，几天内就要截止了。于是它就会理解我的自然语言，帮我把这个写成一个非常清晰的文档，记录在我的服务器上。然后我让它在我的服务器上部署了一个只有我自己能看的网页，那个网页就是我自己可以交互使用的——比如说点进去、修改一些东西，或者说标注完成或者更改时间。就像市面上的那些 To Do List App 那种感觉，而且因为是网页，所以可以同时在电脑和手机上面去使用。所以有什么想要快速记录的时候，我只要用自然语言，就是发语音给我的这个 AI 助手，它就会帮我记好，网页端的 To Do List 也是实时同步的。

那么这样一整个功能，包括规划，包括这个网页的搭建，其实一共下来都花不了几块钱，而且基本上就是一次性解决对话下来就完成了。如果你想抠细节的话，就像我自己去设计的是稍微有点设计风格的，那就可以跟它反复再搞一下，但就这么反复搞几轮都真的花不了几块钱。而普通一个 App 随便开个月费就几十块钱没了。而这个 To Do List 我是真的每天在用，非常好用，比用过以前用过的任何一个 To Do List 的 App 都要好。因为它是我自己做给自己用的，所以是真正了解我自己有什么痛点的。

**小功能二：个性化新闻推送**

还有一个功能就是个性化每天给你做新闻推送。那配置的方式也很简单，就是通过聊天。比如说我想了解 AI 相关的、科技相关的内容，然后需要用到什么信息源，我就跟它说嘛。说完它就给我配置好了。配置好之后每天早上和晚上六点都会给我发一份当时可以搜到的最棒的五个新闻。我配置了几个感兴趣的领域之后，现在就基本上不刷社交媒体了，新闻软件什么的也很久没打开了，已经完全不用看任何 App 上的推送消息了。我现在看的虽然也是三手消息，但已经比八手消息好很多了，也能去掉很多无所谓的噪音。

**小功能三：日志和思维记录**

还有一个就是日志和思维记录。因为我现在真的所有创作和思考，都是通过 Hermes Agent 在我的服务器上完成的。所以和它的对话非常有价值，不仅仅是对我个人而言，对于整个人类物种而言——里面我一整天做的事情、解决了什么问题、思考了什么。比如就说「今天晚餐吃什么」，还有「人类一天最多能喝多少豆浆」——就是豆浆的激素对男性会不会真的不太好，还有「出门 30 分钟要不要关空调」那样更省电，这种反正就是很神奥的一些东西。

那么每天我跟它说的所有话，我都会让它在半夜把那一天发生的所有会话全都提取出来，然后按照话题总结成为一天高度概括型的 Daily Log，第二天早上发给我。由于我一天跟它对话量真的非常庞大，即便是这样一种高度概括的日记，每天至少也是五千字左右。读起来还真的挺有感触的，就是能看到我昨天一天也是非常有生产力。

**高级功能：技能搭建工具**

还有一个比较细的，就是一个技能搭建工具。虽然一开始说 Hermes 它是会自己总结 Skill，然后自己去调用这些 Skill。但对于我个人而言，它调用的方式我觉得它自动化程度有点太高了，就是不那么可控。所以我是自己管理的一个技能库，到现在大概是有 40 多个技能。我自己日常所有的生产和自己也用到的技能，都是我自己审阅、自己创作，然后把他们放到这个技能库里，再通过一个 index 去管理，最后把它部署在我自己的服务器上，用网站去调用和搭建——就跟那个 To Do List 的一样。于是我就[可以]精确地选择哪些 Skills，然后把它们用什么方式做串联起来，[就]可以非常精确地控制完成一整个工作流，而不让 Hermes 它自己野蛮生长。这个工作流我觉得还是挺有价值的，如果你感兴趣的话我就可以专门做一个视频讲一下。

**核心功能：创作工作流**

那上面讲的都是比较小的功能、民用功能。那接下来最重要的当然就是我的整个创作。我以前用的软件就是 Obsidian 和 Notion，现在已经很久没有再打开了。以前写的几千个 MD 文档现在都已经丢进服务器。

想要做视频或者写文章的时候，如果有灵感我就直接语音输入。如果脑子里的想法还不太清晰，我会通过一个特定的技能跟它进行深入聊天，再调用一系列技能，把我跟它聊天过的所有内容全都搜集起来去做成一个大纲。在创造这个大纲的过程中，它会去读取我所有的日志和想法，在里面找到相关的所有内容，并整合进大纲。这样一个大纲通常要改的地方就不多了，因为是一边聊一边产生的东西嘛，所以我是深度参与整个创作的。

然后呢我会让它把这个脚本同时写成英语的，因为我现在同时有经营一个英语的频道和一个中文频道。还有一个我参与了的那个主频道，这也是真的因为有了 Hermes Agent 才能办到的事情。同时我还会让它根据我的脚本去配图，那配图的流程是开头提到了，我这里就不重复了，反正就是框框一通干，效率高得不得了。

**核能功能：公司事务助手**

除了创作方面的功德之外，它还管了我的公司助手——反正事务的处理，那些乱七八糟的报表。每年都有新规，自己去查、去做那些报表真的头很大的。它去查的结果比自己查的要精确多了，而且它对那些专业数据的理解比自己要准确多了。最重要的是，它可以直接从我提供的所有资料里面，计算出报表的哪个空里面具体要填多少的数值。如果是我自己去处理的话，每个季度都有好几天，不仅是消耗我自己的精力，我自己都不一定处理得好——说实话。而这件事情呢，如果你要去雇人的话，那这个花费是 API 头肯的费用——你后面要加大概三个零。

**花费**

那既然说到花费呢，我这边也展开一下。前面是说过 Hermes 它是一个壳。那内部的模型我目前自己用的就是 DeepSeek V4 Flash。我觉得它的革命意义就在于，它用一个非常合理的价格，提供了一个相当有能力的模型。这正好跨越了两个临界点：第一个是它的价格低到我真的就随便造、不心疼；第二个临界点就是，它的能力配上后面随便的——可以完成我给它 95% 甚至 99% 的任务。它完成不了的，我一个月花一两万去雇一个专[门]的员工大概率也完成不了。那要再多雇几个员工或者雇一个更高级的员工，那就变成我给员工打工了。

上面其实所有事情，还有其实还有很多小东西我没有讲的，还有一些我正在开发中的，我都还没有提到。这些东西全都加起来，我可以给你看一下这两个星期稳定运行下来，我的花费大概是这么一个情况——就纯粹我那个服务器里面的 Hermes Agent 的 API 调用，平均下来一天是四块钱左右。服务器本身它一个月要 60 块左右——60 多。但一个服务器里面不是说只能部署一个 Hermes Agent，你还能部署 n8n 什么的。反正我觉得个人有一个服务器还是非常方便的，这个如果你感兴趣，我也可以做一些视频跟大家分享。

**总结**

总的来说就是一个每天只用花四块钱的、超级智能的全能秘书。它知识面比我广，技能上比我深，脾气还好得不得了。但我个人倒也没有什么危机感，因为我知道我们人终究是无法被替代的。因为我是自主，我是有意愿，我是有风格有审美的。更重要的是我一直以来搭建的跟真的人之间的信任关系，这也是为什么我有重新开始做这种聊天了解[的内容]，因为我觉得我跟你们之间这种真实的、真诚的交流，会是接下来时代里面最宝贵的东西之一。
