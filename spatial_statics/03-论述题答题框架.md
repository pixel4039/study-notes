# 论述题答题框架

> 结构：8 个章内对比题（~300字框架）+ 4 个跨章主线题（~800字框架）+ 3 个应用场景题。
> 用法：先看【骨架】能不能自己口述展开；展开不了再读【要点】。所有内容以课件为准。

## A. 章内对比题（300 字级）

### A1. 监督分类与非监督分类的异同（第3讲）

【骨架】定义各一句 → 四维对比 → 各举一例 → 一句适用建议。
【要点】
- 定义：监督=已知部分样本类别及先验知识训练模型，"有目的"分类；非监督=不施加先验，凭数据特征分布"盲目"分类（一般认为即聚类）。
- 对比四维：①先验知识（要/不要）②学习方式（先学习后分类/边学习边分类）③前置工作（须圈定性质单一的训练集/无）④结果含义（类别有语义/仅区分、含义需后续分析）。
- 例：决策树、朴素贝叶斯 vs K-means、层次聚类。
- 收尾：有可靠标注样本且类别明确 → 监督；探索未知结构 → 非监督。

### A2. K-均值聚类与层次聚类的比较（第3讲）

【骨架】各自原理两句 → 五维对比表 → 适用场景。
【要点】原理：K-means 用类中心表征类别、"分配—重算中心"迭代；层次聚类靠相继合并（聚集式）或分割（分割式）构造树。
对比：①是否预先给 k（要/不要）②输出（单一划分/层次树，多粒度）③复杂度（低/很高）④对离群点（非常敏感/相对稳）⑤适用形态（成团分布/更灵活）。
收尾：大数据+类数已知 → K-means；要层次结构或类数未知 → 层次聚类。

### A3. 三大相关系数（Pearson/Spearman/Kendall）比较（第4讲）

【骨架】各自计算基础一句 → 四维对比 → 选用规则。
【要点】计算基础：原始值离差乘积 / 秩差 / 样本对一致性。
对比：①数据水平（间隔比例/次序即可×2）②正态要求（要/不要×2）③关系类型（仅线性/单调含非线性×2）④极端值（敏感/影响很小×2）。
补充：近似线性时 Spearman≈Pearson；多变量场景另有偏相关（控制第三变量）、复相关、典型相关。
选用规则：数据正态+关系线性 → Pearson；偏态、有序数据、担心离群点 → Spearman/Kendall。

### A4. 全局与局部空间自相关（第4讲）★最可能考

【骨架】总定义 → 两级对比 → 各指标点名 → 互补关系收尾。
【要点】
- 全局：单一值概括整个研究区域，判断某现象是否存在空间聚集——Moran's I（正/负/无自相关判读+Z检验，E(I)=−1/(n−1)）、Geary's C、Getis-Ord G（能分高值聚集/低值聚集）。
- 局部：每个空间单元一个值，探究属性变化是平滑（均质）还是突变（异质），定位热点/高发区——局部 Moran's I（LISA，可识别高高/低低/高低离群）、Gi\*（Z>0 显著=热点，Z<0 显著=冷点）。
- 互补：全局说"有没有"，局部说"在哪里"；全局会掩盖局部差异，局部弥补全局不足。
- 例证：深圳 TAZ 人口密度——全局 I 显著为正 → 局部 Gi\* 出热冷点图。

### A5. Moran's I 与 Getis-Ord G 的分工（第4讲）

【骨架】两者思想各两句 → 关键差异 → 组合使用。
【要点】I 比较邻近位置观测值**与均值偏差的乘积**（相关思路）；G 设定距离阈值后直接用邻近观测值**乘积**（量值思路）。
关键差异：I 的正值只说明"相似值聚集"，**分不清是高值扎堆还是低值扎堆**；G 的 Z 为正=高值聚集、为负=低值聚集。
组合：先用 I 确认聚集存在，再用 G/Gi\* 判别冷热并定位。

### A6. 空间滞后模型（SLM）与空间误差模型（SEM）的区别与选择（第5讲）★最可能考

【骨架】共同动机 → 两模型结构与含义 → 判词 → LM 选型流程。
【要点】
- 动机：空间依赖使观测不独立，违反经典回归假设；残差 Moran's I/Geary's C 显著说明缺空间处理（残差高低值空间聚集=缺关键变量）。
- SLM：y=ρWy+Xβ+ε，邻居的**因变量**进方程——扩散/溢出机制（结果传染）；ρ 越接近 1 相邻因变量越相似。
- SEM：y=Xβ+ε，ε=λWε+μ，空间相关藏在**误差**里——遗漏了空间聚集的未观测变量（共享隐因）。
- 选型：先 OLS → 算 LM-lag/LM-error 及 Robust 版本 → 哪个显著用哪个（都显著看 Robust）。案例：美国南部犯罪数据 Robust LM-lag 显著 → 选 SLM，R² 0.104→0.159，残差自相关消失。
- 共同点收尾：都是全局模型、都不能 OLS 估计（用 ML/GMM）。

### A7. 全局回归与地理加权回归（OLS/SAR vs GWR）（第5讲）

【骨架】全局模型的局限 → GWR 原理 → 技术三件套 → 实证效果。
【要点】
- 局限：OLS 乃至 SLM/SEM/SDM 的系数全区一套，假定关系同质，结果只是全区"平均"，掩盖局部特性——对付不了**空间非平稳性**。
- GWR：把地理位置嵌入回归参数 βk(u_i,v_i)，每个采样点一套系数；假设参数是连续表面、邻近位置参数相似，用局部子集做加权最小二乘（WLS）。
- 三件套：空间权函数（高斯、双重平方等距离衰减函数）；带宽（过大→偏差大，过小→方差大；CV/AIC 选）；固定 vs 自适应带宽（分布不均用自适应）。
- 实证：膀胱癌案例 R² 0.25→0.52，且系数空间分布揭示局部负相关；佐治亚案例 0.24→0.30。金句：GWR 能更大限度地挖掘和展示空间异质性。

### A8. IDW、样条与克里金插值方法对比（第7讲）★经典题

【骨架】三者原理各两句 → 五维对比 → 选用建议。
【要点】
- IDW：距离倒数幂加权平均，权重人为指定（p 常取 2）；极值只出现在采样点，易生牛眼。
- 样条：分段函数+边界导数连续，最小曲率薄板；精确、光滑、可超出样本值域；适于平缓渐变面（高程、气象）。
- 克里金：由变异函数（数据自身空间自相关结构）确定权重的线性无偏最优估计；带预测方差（误差估计）。
- 五维：①权重/形态来源（距离公式/光滑约束/变异函数）②是否精确（是/是/普通克里金是）③能否超出观测值域（否/能/能）④误差估计（无/无/**有**）⑤前提（样点均匀较密/变量平缓/满足平稳假设+需结构分析建模）。
- 建议：快速直观 → IDW；平缓表面制图 → 样条；采样贵、要精度与不确定性 → 克里金（有趋势用泛克里金、偏态用对数正态、求超标概率用指示、多变量用协同）。

## B. 跨章主线题（800 字级）

### B1. 论空间依赖与空间异质性：概念、度量与建模应对 ★全课程灵魂题

【骨架】定义与理论来源 → 度量工具 → 对经典方法的挑战 → 两条建模路线 → 总结。
【展开】
1. **概念**（第1讲）：空间依赖=近处实体更相似（Tobler 1970 第一定律）；空间异质性=空间关系参数随地点变化（空间非平稳性）。前者说"值相关"，后者说"规律本身变"。
2. **度量**（第4讲）：依赖→空间自相关指标——全局（Moran's I、Geary's C、G）给全区一个数，局部（局部 Moran、Gi\*）逐单元定位热冷点；异质性→局部指标与后续 GWR 系数面。
3. **挑战**（第2、5讲）：经典统计（t/F 检验、OLS 回归）假设样本独立、关系同质；空间数据两条全违反——残差空间聚集、全局系数只是"平均"。
4. **应对**（第5讲）：依赖 → 空间自回归家族：SLM（ρWy，结果传染）、SEM（λWε，共享隐因）、SDM（+WXδ），LM 检验选型，ML/GMM 估计；异质 → GWR：位置嵌入参数、WLS+空间权函数+带宽，案例 R² 显著提升并揭示局部反转关系。
5. **升华**：插值同理——克里金正面利用空间依赖（变异函数）生产信息；"依赖是资源（可借邻居的信息），异质是警告（别用一套参数打天下）"。

### B2. 论空间权重矩阵：一条贯穿全课程的线索 ★课件自问过"还有哪些指标用了空间权重矩阵"

【骨架】W 是什么 → 怎么构建 → 在四处出场 → 构建选择的影响 → 总结。
【展开】
1. **定义**（第4/5讲）：n×n 矩阵，w_ij 描述单元 i、j 邻近程度，对角线 0；是空间关系进入统计模型的接口。
2. **构建**（第4讲）：邻接法——Rook（共边）/Bishop（共顶点）/Queen（边或顶点）；距离法——欧氏/球面距离及其倒数、阈值（G 统计量的 d）。第6讲提醒：真实城市里还可用**网络距离**。
3. **四处出场**：①空间自相关——Moran's I、局部 Moran、G/Gi\* 的 w_ij；②空间自回归——SLM 的 ρW₁y、SEM 的 λW₂ε（W 即空间滞后算子，Wy=邻居值的加权平均）；③GWR——空间权函数（高斯、双重平方…）+带宽生成的对角权阵；④克里金——变异函数本质上也是"距离→权重"的结构化版本（由数据估计而非人为指定）。
5. **影响与总结**：W 的选择直接改变 I 值、模型系数与插值结果；"空间统计的一切方法，都是给'邻居'下定义再让邻居发言"。

### B3. 从经典统计到空间统计：这门课在修什么 bug ★适合"谈谈你对空间统计的理解"类开放题

【骨架】经典统计的独立性假设 → 空间数据四特征与两性质 → 逐讲的"空间化改造" → 应用价值。
【展开】
1. 经典统计（第2讲）：描述统计、正态、t/F 检验、OLS——共同前提是**样本独立同分布**。
2. 空间数据（第1讲）：空间/属性/时间/多尺度四特征；Tobler 定律使邻近样本天然相关——独立性假设系统性失效（80% 数据与空间位置相关，问题无法回避）。
3. 逐讲改造：相关分析→空间自相关（第4讲，"和邻居的相关"）；回归→SAR/GWR（第5讲，把 W 和位置写进模型）；插值→克里金（第7讲，用变异函数刻画并利用相关）；距离→网络距离与可达性（第6讲，度量空间本身空间化）。
4. 两面性：空间相关既是**麻烦**（假设失效、检验失真）也是**资源**（凭邻居信息插值预测——克里金、SLM 的预测力都来自它）。
5. 收尾：John Snow 案例首尾呼应——把数据放到空间上，才能发现非空间方法看不见的规律。

### B4. 空间分析方法体系在城市研究中的综合应用 ★偏应用型论述/附加题

【骨架】给一个城市问题 → 按课程主线串一遍方法 → 每步产出什么。
【展开】（以"城市公共服务（如学校/医疗）优化"为例，深圳背景）
1. **摸家底**：描述统计+统计图刻画各区人口/设施分布（第2讲；深圳各区人口 CV=84.4%，高度不均）。
2. **找结构**：聚类给街区分组画像（第3讲 K-means）；空间自相关找需求热点——全局 Moran's I 确认聚集，Gi\* 定位热冷点（第4讲，深圳 TAZ 人口密度案例）。
3. **找原因**：回归解释设施利用/需求的影响因素；残差空间聚集 → SLM/SEM；关系随区位变 → GWR 揭示局部差异（第5讲）。
4. **算服务**：网络分析——缓冲区 vs 网络服务区，Dijkstra 最短路径，学校选址（定位）与消防站辖区（分配），可达性三大方法评估公平性（第6讲，义务教育选址正是课件案例）。
5. **补空白**：对未观测地点插值——环境变量用克里金/LUR（第7讲）。
6. 总结句：空间统计提供了"描述→诊断→解释→优化→预测"的完整工具链。

## C. 应用场景题（简答/附加题级）

### C1. 空气污染制图该选哪种插值？
答题链：监测站稀疏（点数据）→ 需连续面 → 候选：IDW（快但牛眼、无误差估计）、趋势面（只有宏观趋势）、LUR（引入土地利用/交通协变量，城市内变异刻画好——课件明示是模拟城市大气污染分异的通用模型）、克里金（变异函数+误差图，可配协同克里金借道路/人口变量）。结论：城市尺度 LUR 或克里金为主，IDW 做快速初版。

### C2. 犯罪/疫情热点分析怎么做？
答题链：点事件 → 核密度估计（KDE，带宽权衡）出密度面；或聚合到空间单元 → 全局 Moran's I 验证聚集 → Gi\* 出分置信度热点图（0.1/0.05/0.01）→ 若要解释成因，SLM/SEM（第5讲课件即犯罪案例）。历史锚点可提 John Snow：热点分析的鼻祖操作。

### C3. 如何评价一个分类/聚类结果的好坏？
答题链：分类——留出测试集 → 混淆矩阵 → 总体精度 → 指出类别不平衡陷阱（全猜多数类 99%）→ Kappa 扣除偶然一致；聚类——类内最相似类间最不相似 → RMSE（只看类内、算得快）→ DBI（类内+类间，越小越好）；再提聚类结果含义需后续分析确定。
