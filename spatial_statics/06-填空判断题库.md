# 填空 + 判断 自测题库

> 用法：拿张纸盖住括号里的答案，逐题自测。错的题在题号上画圈，凌晨只重做画圈题。
> 填空 70 题按讲次排；判断 40 题故意混入"看起来对的错话"，全部来自课件易错点。

## 一、填空题（70 题）

### 第 1 讲

1. 空间分析是 GIS 区别于______的重要标识。（IS／信息系统）
2. 1854 年，______在伦敦霍乱地图上标出病死者居住位置，锁定了被污染的水井。（John Snow／约翰·斯诺）
3. Tobler 地理学第一定律提出于______年，核心是"任何事物都相关，______的事物比远处的更相关"。（1970；邻近／近处）
4. 空间数据的四大基本特征是：空间特征、属性特征、______特征和______特征。（时间；多尺度）
5. 空间特征中的空间关系包括拓扑关系、______关系和______关系。（方位；距离）
6. 对空间依赖的定量描述称为______。（空间自相关）
7. 空间异质性又称空间______性。（非平稳）
8. 空间分析发展史五阶段：地图分析 → ______ → 计量地理学 → ______ → 空间大数据分析。（地统计；地理计算）
9. 大约______%的数据与空间位置相关。（80）
10. 度量空间依赖的常用方法有 Moran's I、Geary's C、Ripley's K、______指数和半变异函数。（Join count）

### 第 2 讲

11. 1 阶中心矩是期望，2 阶中心矩是______，3 阶是______，4 阶是______。（方差；偏度；峰度）
12. 右倾（右偏）分布中：______ ＞ ______ ＞ ______。（均值；中位数；众数）
13. 变异系数 CV ＝ ______ ÷ ______，是无量纲量；当______接近 0 时 CV 无效。（标准差；均值；均值）
14. 正态分布的 3σ 准则：μ±σ 内约______%，μ±2σ 内约______%，μ±3σ 内约______%。（68.26；95.44；99.74）
15. 正态分布密度曲线的拐点位于 x = ______。（μ±σ）
16. 正态分布中 μ 是______参数，σ 是______参数。（位置；尺度）
17. 首次导出正态分布的是______（1809），"正态分布起源于______分析"。（高斯；误差）
18. 拒绝了本该接受的原假设，犯的是______类错误（又称______错误），概率记为______。（Ⅰ；弃真；α）
19. 同时减少两类错误的唯一方法是______。（增加样本容量）
20. 检验两总体均值差异用______检验，检验方差齐次性用______检验。（t；F）
21. 双总体 t 检验的前提：样本来自______总体，且两总体______相等。（正态；方差）
22. 判断"是否有增长"这类强调方向的检验应采用______侧检验。（单）
23. 盒图体现五数概括：最小值、______、中位数、______、最大值。（下四分位数 Q1；上四分位数 Q3）
24. 判断两个数值变量是否关联、识别离群点，最常用的统计图是______图。（散点）

### 第 3 讲

25. 聚类的目标：相同类别之内相似性越______，不同类别之间差异性越______。（高；大）
26. 常用相似性度量有欧氏距离、______和______。（相关系数；余弦相似性）
27. Z-Score 标准化后各指标均值为______、标准差为______；极差标准化后值域为______。（0；1；[0,1]）
28. K-均值聚类必须预先输入______，且对______样本非常敏感。（类别数 k；异常／离群）
29. 层次聚类的两种形式：______层次法（自底向上）和______层次法（自顶向下）。（聚集；分割）
30. DB 指数最小为______，越______聚类质量越好。（0；小）
31. 决策树构建三步骤：特征选择、______、______。（新增节点／节点分裂；剪枝）
32. 决策树划分准则是使样本子集______更高，如______减少。（纯度；信息熵）
33. Kappa = (P₀ − Pe)/(1 − Pe)，其中 Pe 是______的样本比例。（偶然一致性）
34. Kappa 接近 0 说明分类结果可能是______造成的。（偶然）
35. 监督分类是先______后______；非监督分类是边______边______。（学习；分类；学习；分类）
36. 模型在训练样本上准确度高、对其他样本准确度低的现象称为______。（过拟合）

### 第 4 讲

37. Pearson 相关系数适用条件：变量为间隔或比例水平数据、符合______分布、只能衡量______相关。（正态；线性）
38. n=2 时 Pearson 相关系数 r 恒等于______，故需对 r 进行______检验。（1；显著性（t））
39. Spearman 相关系数基于样本数据的______（秩）计算，能反映变量间______关系的强弱。（等级；单调）
40. Kendall's τ = （一致对数 − ______）÷ 总对数。（不一致对数）
41. 控制其他变量影响后两变量的直线相关程度称为______相关；零阶偏相关即______。（偏；普通相关）
42. 两地理单元有共同边界为______相邻；有公共顶点为______相邻；边或顶点皆可为______相邻。（Rook；Bishop；Queen）
43. 全局空间自相关用______个值反映整个区域；局部指标为______空间单元各算一个值。（单一／一；每个）

48. 局部空间自相关指标能弥补______分析掩盖局部差异的不足。（全局（空间自相关））
49. 深圳案例以______（TAZ）为基本空间单元、人口密度为观测值，邻近性取欧氏距离的______。（交通分析小区；倒数）

### 第 5 讲

50. "回归"一词源于______的遗传学研究：后代身高有向______回归的趋势。（高尔顿（Galton）；平均（均值））

62. 空间回归选型用______检验（服从卡方分布）：Robust LM-lag 显著选______，Robust LM-error 显著选______。（LM（拉格朗日乘数）；SLM；SEM）

## 二、判断题（40 题，注意理由）

### 第 1-2 讲

1. 空间自相关和空间依赖是同一个概念的两种叫法。（**错**——依赖是性质，自相关是对它的定量描述）
2. Tobler 第一定律说：近处的事物是相同的。（**错**——是"更相关"，不是相同）
3. 空间数据的基本特征包括空间、属性、时间三类。（**错**——还有第四个：多尺度特征）
4. 均值是集中趋势最主要的测度值，且对极端值稳健。（**错**——均值对极端值很敏感）
5. 中位数反映灵敏，少量数据变化一定会改变中位数。（**错**——可能不产生影响，反映不灵敏）
6. 众数一定存在，但不一定唯一。（**错**——不一定存在，存在也不一定唯一）
7. 方差的单位与原始数据一致。（**错**——方差是平方单位；标准差才一致）
8. 变异系数可用于不同单位数据之间离散程度的比较。（**对**——无量纲正是它的价值）
9. 左倾分布中众数＞中位数＞均值。（**对**）
10. 显著性水平 α 是愿意犯Ⅱ类错误的最大概率。（**错**——是Ⅰ类（弃真）错误）
11. 样本量固定时，减少Ⅰ类错误往往使Ⅱ类错误增加。（**对**）
12. F 检验常作为选择何种均值检验（t 检验形式）的基础。（**对**——先验方差齐不齐）
13. 置信水平 95% 表示"这个区间有 95% 的概率包含真值"。（**错**——是重复构造区间时包含真值的长期比例）

### 第 3 讲

14. 聚类需要样本集合的先验知识。（**错**——聚类无需先验知识）
15. 聚类结果直接给出每个类别的实际含义。（**错**——含义需后续分析确定）
16. K-means 的终止条件只能是均值点不再变动。（**错**——还可以是 DBI 达阈值或最大迭代次数）
17. 层次聚类必须预先指定类别数。（**错**——不用，事后在树上选；要预先给 k 的是 K-means）
18. 决策树的构建要求样本满足正态分布。（**错**——决策树无统计分布要求）
19. 决策树对训练样本的微小变动不敏感。（**错**——敏感性很高）
20. 类别不平衡时总体分类精度可能虚高，因此需要 Kappa 系数。（**对**）
21. Kappa 接近 −1 说明分类结果可能是偶然的。（**错**——接近 0 才是偶然；−1 是严重不一致）
22. 一般认为非监督分类就是聚类。（**对**——课件原话）

### 第 4 讲

23. 相关关系可以用数学公式准确表达。（**错**——那是确定关系；相关关系是非完全确定的趋势）
24. 两变量相关显著即说明存在因果关系。（**错**——相关≠因果）
25. Pearson 相关系数 r=0 说明两变量没有任何关系。（**错**——只说明无明显线性关系）
26. r 从 0.3 增大到 0.6，说明相关程度增加了一倍。（**错**——相关系数不是等距测量）
27. Spearman 相关系数要求两变量服从正态分布。（**错**——无需正态，这是 Pearson 的要求）
28. Spearman 相关系数受极端值影响很小。（**对**）
29. Queen 相邻要求两个单元既有公共边又有公共顶点。（**错**——有边**或**顶点即可）
30. Moran's I = −0.8 说明该区域不存在空间自相关。（**错**——是显著负自相关（高低相间））
31. Moran's I 为正且显著时，能判断是高值聚集还是低值聚集。（**错**——I 分不清高低聚，要用 G/Gi\*）
32. Moran's I 的随机期望是 0。（**错**——是 −1/(n−1)）
33. 全局指标用单一值概括全区，会掩盖局部差异。（**对**——所以需要局部指标）

### 第 5-7 讲

34. 空间滞后模型（SLM）是把空间相关性放进误差项的模型。（**错**——那是 SEM；SLM 把邻居的因变量 ρWy 放进方程）
35. SLM、SEM、SDM 和 GWR 都是全局空间回归模型。（**错**——前三个是全局，GWR 是局部（每点一套系数））
36. GWR 的带宽越小，估计的偏差越大。（**错**——带宽小→方差大；带宽大→偏差大）
37. 空间滞后模型的参数可以用普通最小二乘法估计。（**错**——须用 ML 或 GMM）
38. IDW 插值的极值只可能出现在采样点上，样条插值则可超出样本值域。（**对**）
39. 克里金与 IDW 都是加权平均，区别在于克里金的权重由变异函数确定且能给出误差估计。（**对**）
40. 变异函数中的块金值反映变量的影响范围。（**错**——块金值反映随机性；影响范围是变程）
