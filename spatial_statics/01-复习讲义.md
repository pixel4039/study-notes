# 第 1 讲 导论：空间分析是什么

## 【1分钟速览】

空间分析研究**空间实体及其空间特征**，是 GIS 区别于普通信息系统（IS）的标志。空间数据有**空间、属性、时间、多尺度**四类基本特征。两大核心性质：**空间依赖**（近者相似，Tobler 第一定律）和**空间异质性**（不同地方规律不同）。发展史五阶段：地图分析 → 地统计（1930s）→ 计量地理学（1950-60s）→ 地理计算（1990s）→ 空间大数据分析（2011-）。经典案例：John Snow 1854 伦敦霍乱地图。

## 1.1 起点案例：John Snow 与伦敦霍乱（1854）

1854 年 8-9 月伦敦霍乱流行，当局找不到病因。**John Snow（流行病学之父）**在 1:6500 地图上标出**每个霍乱病死者的居住位置**，叠加道路、房屋、**饮用水机井**，得到病死者居住分布图——死亡病例围绕宽街（Broad Street）水泵聚集，据此锁定污染水源。

**这个案例为什么是空间分析的"创世神话"**：它示范了空间分析的完整逻辑链——
**把属性数据（病例）落到空间位置上 → 观察空间分布模式（聚集）→ 推断背后的空间过程（水井传播）→ 支持决策（关闭水泵）**。
答任何"空间分析有什么用"的题，都可以用这条链收尾。

> 📜 **历史锚点**：1854 年细菌学说尚未确立，主流理论是"瘴气说"（脏空气致病）。Snow 的地图之所以赢，是因为**空间证据**指向了瘴气说解释不了的模式：病例围绕特定水井聚集，而不是沿"脏空气"均匀弥散。输家（瘴气说）输在无法解释空间分布的**不均匀性**。——记忆钩子：空间模式能证伪非空间理论。

## 1.2 空间分析的定义与地位

课件给出多家定义（了解即可，考试能复述一种+说出共性就够）：

| 学者 | 定义要点 |
|------|---------|
| Unwin | 首次研究空间分析框架（空间数据操作/数据分析/统计分析/建模） |
| Goodchild | 空间查询和空间数据分析，从 GIS 目标之间的空间关系中获取派生信息和新知识 |
| Bailey, Hanning, Ripley | 从空间统计学角度对空间数据进行运算与分析 |
| 郭仁忠、李德仁 | 基于空间对象及空间布局的地理数据分析，目的在于提取和传输空间信息 |

**共性**（简答可用）：研究对象是**空间实体及其空间特征**；内容包括空间分析**理论、方法、应用**三层；目标是从空间数据中提取模式（patterns）、描述（descriptions）与洞见（insights），走向空间智能。

**一句话地位**：**空间分析是 GIS 区别于 IS 的重要标识**——普通信息系统只能存、查、管，GIS 因为有空间分析才能"发现新知识、支持决策"。（这句课件重复了两遍，极可能考。）

**学科关系**：统计学（理论基础）+ GIS/计算机（技术基础）→ 空间分析 → 应用于地理学、城市科学、环境、公共卫生等。

## 1.3 发展史：五个阶段

| 阶段 | 时间 | 关键词 | 贡献与局限 |
|------|------|--------|-----------|
| 地图分析 | ~1900 起 | 图上量测 | 揭示制图对象时空特征及相互关系；获得分布规律、要素联系、动态变化、综合评价 |
| 地统计 Geostatistics | 萌芽于 1930s | 统计量描述地理要素 | 把统计学引入地理学，给定性描述带来定量化；但**依靠人工计算，只能对少数要素统计** |
| 计量地理学 | 1950s 末-60s | 数学方法+计算机 | 定量判断弥补定性描述；数学模型刻画地理现象；模拟预测代替现状分析；趋势推导代替简单因果 |
| 地理计算 GeoComputation | 1990s | 高性能计算 | 融合计算机科学、地球信息科学、数学统计；Openshaw(2000)：用高性能计算解决目前不能解决甚至未知的空间问题 |
| 空间大数据分析 | 2011- | 大数据 | 2011 年 Science 专刊宣告大数据时代到来；**80% 的数据与空间位置相关**；数据含空间、时间、属性三个基本特征 |

计量地理学自身的三阶段（可能出简答细节）：
1. **50 年代末-60 年代末**（初期）：统计学方法引入地理学
2. **60 年代末-70 年代**（中期）：多元统计方法 + 计算机技术
3. **70 年代末至今**（成熟）：与运筹学、系统科学、GIS 技术结合

> 📜 **历史锚点**：为什么"地图分析"被计量地理学取代？地图分析只能靠**眼睛**读图，结论无法复现、无法检验显著性。计量革命的胜负手是**可重复的定量判断**。而计量地理学后来又被批"只见数字不见人"，地理计算用算力把它复活——每一代都是"上一代的短板"驱动的。记发展史别背年份孤岛，背"每一代解决了上一代什么痛点"。

**空间大数据分析过程**（六环节，认得即可）：数据获取 → 存储 → 计算 → 挖掘 → 呈现 → 安全。

## 1.4 空间数据及其四大基本特征 ★名词解释高危

**空间数据（Spatial data）**：对现实世界中空间事物和现象**时空特征和过程**的**抽象表达和定量描述**。

抽象链条：现实世界 →（抽象化：场模型/要素模型）→ 概念模型·空间实体 →（数字化：栅格/矢量数据模型）→ 空间对象。

四大基本特征（**必背**）：

| 特征 | 内容 |
|------|------|
| ① 空间特征 | 空间分布位置、几何形状、空间关系（**拓扑关系、方位关系、距离关系**） |
| ② 属性特征 | 名义、次序、间距、比率属性（定性→定量的四个层级） |
| ③ 时间特征 | 空间实体随时间变化的特征 |
| ④ 多尺度特征 | 因兴趣领域、研究过程、研究范围、度量标准不同而呈现多尺度；伴随数据综合 |

## 1.5 空间依赖 vs 空间异质性 ★全卷最爱考的对比之一

| | 空间依赖 Spatial Dependence | 空间异质性 Spatial Heterogeneity |
|---|---|---|
| 定义 | 空间上距离相近的实体之间的相似性比距离远的实体间的相似度大 | 描述空间关系的参数在研究区域的**不同地方是不一样的**，但在区域局部其变化是一致的 |
| 理论来源 | **Tobler（1970）地理学第一定律** | 空间非平稳性 |
| 定量化 | **空间自相关**（对空间依赖的定量描述） | 局部指标 |
| 尺度 | 全局观测：对研究区域整体给一个参数/指数 | 局部尺度：给每个观测点等量的参数/指标 |
| 常用方法 | **Moran's I、Geary's C、Ripley's K、Join count 指数、半变异函数** | **局部 Moran's I、局部 Geary's C、空间非平稳性指数** |
| 相关概念 | 空间自相关 | 空间非平稳性、各向同性 |

**Tobler 地理学第一定律（1970）**原文意涵：Everything is related to everything else, but near things are more related than distant things——**任何事物都相关，近处的事物比远处的更相关**。它是空间自相关、空间插值（IDW、Kriging）、空间回归的共同理论根基——跨章论述的万能线头。

**记忆抓手**：依赖说"**近朱者赤**"（值随距离相关）；异质说"**橘生淮南则为橘**"（规律本身随地点变）。第 4 讲全局/局部 Moran 对应前者，第 5 讲 GWR 对应后者——伏笔埋好。

## 1.6 空间分析主要内容与应用

课程主线（就是后面 6 讲的目录）：空间数据统计分析（第2讲）→ 聚类分类（第3讲）→ 相关性分析与空间自相关（第4讲）→ 回归与空间回归（第5讲）→ 网络分析（第6讲）→ 空间插值（第7讲）。

典型应用（附加题素材，报菜名即可）：居民活动规律（数量/空间分布/组成）、交通出行与道路流量、城市交通智能化管理、**义务教育段中小学选址**（公共服务均等化，深圳案例——本校老师自己的研究方向，答应用题优先举这个）。

其他案例：全球夜光地图（经济活动的空间代理变量）、**胡焕庸线**（人口地理分界线，空间异质性的经典中国案例）、地形分析。

## 易错点

1. "空间分析是 GIS 区别于 **IS**"——别写成"区别于遥感/测绘"。
2. 空间依赖的定量化叫**空间自相关**；两个词不是同义反复：依赖是性质，自相关是度量。
3. 四大特征别漏"多尺度特征"——最容易忘的第四个。
4. Tobler 定律年份 **1970**（不是 1954，那是霍乱……也不对，霍乱是 1854）。

**本讲在全课程主线中的位置**：总纲——它提出的"空间依赖/空间异质性"二分，是后面所有方法的分水岭：第 4 讲测依赖，第 5 讲 GWR 治异质，第 7 讲插值用依赖。

# 第 2 讲 经典统计学基础

## 【1分钟速览】

三块：**描述性统计**（集中趋势：均值/中位数/众数；离散趋势：极差/方差/标准差/变异系数；分布形态：偏度/峰度）→ **正态分布**（μ 定位置 σ 定形状，3σ 准则 68-95-99.7）→ **统计推断**（参数估计·置信区间；假设检验：H0/H1、两类错误、t 检验比均值、F 检验比方差）。一切"经典"方法都假设样本**独立**——而空间数据恰恰不独立，这就是后面所有"空间 XX 方法"存在的理由。

## 2.1 描述性统计

**定义**：运用制表和分类、图形以及计算概括性数据来**描述数据特征**的各项活动。

三类度量 + 一个统一框架（**矩**）：**1 阶中心矩=期望｜2 阶中心矩=方差｜3 阶中心矩=偏度｜4 阶中心矩=峰度**（这行课件原话，简答装点门面极好用）。

### 集中趋势

| 统计量 | 定义 | 特性/坑 |
|--------|------|--------|
| 均值 | 四种：算数、加权算数、几何、调和 | 最有用的单个量，但**对极端值很敏感** |
| 中位数 | 排序后 50% 位置的数 | **对极值不灵敏**；少量数据变化可能对它无影响（反映不灵敏） |
| 众数 | 出现次数最多的数 | **不一定存在，存在也不一定唯一**；单峰/双峰/多峰 |
| 中列数 | (最大值+最小值)/2 | 快速粗估 |

分位数体系：四分位数 Q1(25%)、Q2(50%)=中位数、Q3(75%)；十分位数 D1…D9；百分位数 P1…P99。

**三者关系**（画个小峰记住）：
- 对称分布：均值 = 中位数 = 众数
- **右倾（右偏）分布：均值 ＞ 中位数 ＞ 众数**（长尾拖着均值往右跑）
- 左倾分布：众数 ＞ 中位数 ＞ 均值

### 离散趋势

| 统计量 | 定义 | 特性 |
|--------|------|------|
| 极差 Range | Max − Min | 只用两个数，粗糙 |
| 半内四分位距 | (Q3 − Q1)/2 | 抗极值 |
| 10~90 百分位距 | P90 − P10 | 同上 |
| 平均偏差 MD | 各数与均值之差的**绝对值**的平均 | |
| 方差 | 各值与均值**离差平方**的平均 | 反映灵敏（每个数都参与）；受抽样变化影响小；**有单位且与原单位不一致**（平方） |
| 标准差 | 方差的算数平方根 | 同方差，但**单位与原数据一致**；可当作不确定性的度量 |
| 变异系数 CV | **标准差 ÷ 均值**，常表示为百分数 | **无量纲**，可比较**不同单位/不同水平**数据的离散程度；**均值接近 0 时失效** |

**CV 为什么存在**（简答高频）：10 英寸的偏差，出现在 1000 英尺的测量里微不足道，出现在 20 英寸的测量里是灾难——绝对离差不可比，除以均值变成相对离差才可比。

课件算例（认得数量级即可）：深圳十区常住人口，方差 1422.5、标准差 37.7 万人、CV = 84.4%——CV 这么大说明深圳各区人口极不均衡（附加题可引）。

### 分布形态

**偏度**（3 阶矩，分布不对称程度、偏的方向）、**峰度**（4 阶矩，峰的尖平程度）。认识即可。

### 常用统计图 ★简答"用什么图判断什么"

| 图 | 用途 |
|----|------|
| 散点图 | 判断**两个数值变量**是否关联；识别**离群点** |
| 直方图 | 单变量分布；**初判是否正态**；分类变量版本叫条形图；数值域切成桶(bin) |
| 盒图 Boxplot | **五数概括**：最小值、Q1、中位数、Q3、最大值；盒长 = IQR |
| 分位数图 | 单变量分布 + 分位数信息 |
| Q-Q 图 | **比较两个分布**是否有漂移（一个分布的分位数对另一个的分位数作图） |

## 2.2 正态分布

**定义**：X~N(μ, σ²)，又叫高斯分布。身高、寿命、成绩、**测量误差**等都近似服从。

概率密度函数认得样子就行；关键读法：**μ 是位置参数（曲线中心在哪），σ 是尺度参数（胖瘦）**。

四条曲线特征（课件原文，简答直接列）：
1. 由 μ 和 σ 决定位置和形状
2. 以均值为中心，**左右对称**
3. 以 X 轴为渐近线
4. **拐点在 x = μ ± σ**

**标准正态分布**：μ=0，σ=1；一般正态可标准化（Z 变换）转过去。

**3σ 准则**（必背数字）：μ±σ 内 **68.26%**，μ±2σ 内 **95.44%**，μ±3σ 内 **99.74%**。

**正态分布的三大作用**：测量数据处理理论的基础、参数估计、假设检验。

> 📜 **历史锚点**：正态分布**起源于误差分析**。伽利略最早提出随机误差 → 棣莫弗(1730s)用二项分布逼近得到密度函数形式 → 勒让德(1805)最小二乘 → **高斯(1809)引入正态误差理论，首次导出正态分布** → 拉普拉斯(1811)用中心极限定理论证。20 世纪初它遭质疑（真实数据拟合欠佳），催生了**卡方、t、F 三大分布的小样本革命**——你下一节学的 t 检验、F 检验就是这场革命的产物。为什么正态还是赢家？中心极限定理：大量独立小误差之和趋于正态——它是"大样本世界的默认分布"。

## 2.3 参数估计（略讲，认得概念）

统计推断 = **参数估计 + 假设检验**（描述性统计之外的另一半）。

- **参数估计**：从总体中抽取随机样本来估计总体分布中未知参数的过程。**估计量**（如样本均值、样本方差这种"公式"）vs **估计值**（代入具体样本算出的数）。
- **置信区间**：由样本统计量构造的总体参数的估计区间。**置信水平**：重复构造多次区间，包含总体参数真值的次数比例（不是"这个区间有 95% 概率包含真值"——是长期频率）。

## 2.4 假设检验 ★本讲考试核心

**逻辑**：先对总体做假设，再用样本信息检验其是否成立。

| 概念 | 定义 | 记忆点 |
|------|------|--------|
| 原假设 H₀ | 多数情况下**提出来就是为了拒绝它**的假设 | 判断新方法是否更优时，先假设"差异不存在" |
| 备择假设 H₁ | 任何不同于 H₀ 的假设，拒绝 H₀ 后备选 | 研究假设——你真正想证明的 |
| **Ⅰ类错误（弃真）** | **拒绝了本该接受的 H₀**，概率记 α | "冤枉好人" |
| **Ⅱ类错误（存伪）** | **接受了本该拒绝的 H₀**，概率记 β | "放走坏人" |
| 显著性水平 α | **愿意犯Ⅰ类错误的最大概率** | 常取 0.05 |

课件例子（瑞德西韦治新冠）：H₀ = 新药无效，H₁ = 新药有效。Ⅰ类错误 = 药本无效却宣布有效（危害大），Ⅱ类错误 = 药本有效却说无效。

**两类错误的关系**（简答高频）：样本容量给定时，减少一类错误会使另一类**增加**；**同时减少两类错误的唯一方法是增加样本容量**。实际中Ⅰ类错误往往更严重，所以优先控制 α。

**单侧 vs 双侧**：只强调"有无差异"不管方向 → 双侧；强调某一方向（如"是否**增长**"）→ 单侧。

### t 检验：比均值

用途：比较两个均值间差异是否显著（H₀：均值相等）。

- **单总体**：样本均值 x̄ 与已知总体均值 μ₀ 比。统计量 t = (x̄ − μ₀)/(s/√n)，服从 t(n−1)。**读式子**：分子是"观测差异"，分母是"均值的标准误差"——t 就是"差异有几个标准误差那么大"。
- **双总体（独立样本）**：两样本均值差 ÷ 合并标准误，自由度 n₁+n₂−2。

**适用前提**（必背）：① 样本来自正态总体；② 双总体要求两总体**方差相等（方差齐次性）**，否则用近似 t 检验。

课件例题套路（六步，简答"简述假设检验步骤"直接搬）：
**① 建立假设 → ② 定单/双侧 → ③ 选显著性水平 α → ④ 计算统计量 → ⑤ 查临界值表 → ⑥ 比较并下结论**。
例：13 岁男孩身高 10 年前 1.51m，现在 200 人均值 1.53m、s=0.073 → t≈3.87 > t₀.₀₅(199)≈1.653 → 拒绝 H₀，身高显著增长。

### F 检验：比方差

用途：**方差齐次性检验**，检验两总体方差差异是否显著（H₀：方差相等）。

统计量 F = s₁²/s₂²，服从 F(n₁−1, n₂−1)。**读式子**：两个样本方差之比；离 1 越远越可疑。技巧：**把大方差放分子使 F>1**，只查右侧临界值。

要点：① 同样要求正态总体；② 检验方差相等时一般**双侧**；③ **F 检验常是 t 检验的前置步骤**——先 F 检验定方差是否齐，再决定用哪种 t 检验。（这个先后关系是经典简答点。）

## 易错点

1. **两类错误方向**：Ⅰ弃真（拒真 H₀）α，Ⅱ存伪（收假 H₀）β——考场最容易写反的一对。
2. 方差单位与原数据**不一致**，标准差**一致**——别写反。
3. CV = 标准差/均值（不是方差/均值），均值≈0 时**无效**。
4. 3σ 三个数：68.26 / 95.44 / 99.74。
5. t 检验比**均值**，F 检验比**方差**；顺序是先 F 后 t。
6. 右偏分布是**均值＞中位数＞众数**（尾巴方向=均值被拖的方向）。

**本讲在全课程主线中的位置**：全课程的"工具箱底层"——但经典统计假设**样本独立**，空间数据天然违反这一点（空间依赖），后面每一讲的"空间化改造"都是在修这个 bug。

# 第 3 讲 聚类与分类

## 【1分钟速览】

**聚类** = 无标签、按相似性自动分组（非监督）；**分类** = 有标签、先训练后预测（监督）。聚类三件套：相似性度量（欧氏距离/相关系数/余弦）→ 标准化（Z-score/极差）→ 算法（K-means 中心聚类、层次聚类）；质量评估用 RMSE、DBI。分类主讲**决策树**；模型评估用**混淆矩阵 → 总体精度 → Kappa 系数**（防"全猜多数类"的假高精度）。

## 3.1 聚类的定义与基本过程

**聚类（Clustering）**：将一组样本组织为不同的类别或层次结构，使得**相同类别之内的相似性越高，不同类别之间的差异性越大**。《现代地理学辞典》版：按一定标准，鉴别地理区域或实体、现象之间的接近程度，并将相接近的归为一类的数学方法。

**为什么需要**：传统分类靠经验和专业知识；数据量变大、要求变高后，凭经验不能进行确切分类——需要算法。

**聚类分析基本过程**（五步，简答直接背）：
1. 选择描述事物**本质属性**的指标，去除无关指标
2. 确定是否需**标准化**，选择标准化方法
3. 选择样本**相似度度量**（距离计算）方法
4. 选择和实施**聚类算法**
5. **结果分析和调整**

## 3.2 相似性度量与标准化

三种常用相似度（对 p 个指标的两样本向量）：

| 度量 | 含义读法 |
|------|---------|
| **欧氏距离** | 特征空间中的直线距离；差的平方和开根号；越小越相似 |
| **相关系数** | 两组指标围绕各自均值的协同波动程度；−1~1 |
| **余弦相似性** | 两向量夹角的余弦；只看方向不看长度 |

**为什么要标准化**（简答高频）：多数相似性度量要求指标**量纲统一**。量纲级别高的指标会**假相扩大**差异，量纲低的会**假相缩小**差异（例如"人口(万)"和"GDP(亿元)"直接算距离，GDP 数字大就霸占了距离）。

两种标准化：

| 方法 | 公式思想 | 结果性质 |
|------|---------|---------|
| **Z-Score（标准差标准化）** | (值 − 均值) ÷ 标准差 | 各指标均值 0、标准差 1 |
| **极差标准化** | (值 − min) ÷ (max − min) | 值域 [0,1] |

## 3.3 聚类质量评估

标准：**类内最相似，类间最不相似**。

- **RMSE**：只反映**类内**相似性（类内到均值点的离散度）；计算简单，一般只在**计算效率要求高**时用。
- **DB 指数（Davies-Bouldin Index）**：综合类内散度与类间距离；**最小为 0，越小越好**。读法：对每个类，找"和它最像的邻类"算一个（类内散+类内散)/类间距的最坏比值，再平均——类内越紧、类间越远，DBI 越小。

## 3.4 K-均值聚类 ★必考算法

**思想**：认为一个类可用**类内样本均值点（聚类中心）**表征；迭代优化中心点。

**流程**（必须能默写）：
1. 输入类别数 **k**
2. 初始化类别均值点（或随机分配样本）
3. 把每个样本**分配到最近均值点**所在类
4. **重新计算**每类均值点
5. 重复 3-4 直到终止条件：**均值点不再变动 / DBI 降到阈值 / 达到最大迭代次数**

**优缺点**（对比题弹药）：
- ✅ 计算复杂度**较低**
- ❌ 需**预先输入 k**
- ❌ 对**异常/离群样本非常敏感**（均值被拽跑）
- ❌ 只适合特征空间中**成团分布**的样本，复杂分布模式（环形、条带）效果差

## 3.5 层次聚类

**思想**：自然类别可能存在**层次结构**；通过一系列相继的合并或分割构造树。

| 形式 | 方向 |
|------|------|
| **聚集层次法**（自底向上） | 从单个样本开始，两两最相似的合并，直到只剩一个类 |
| **分割层次法**（自顶向下） | 全体作一个类，逐次分成最不相似的两块，直到每样本一类 |

课件例：**单连接层次法**——每次在样本-样本、样本-类、类-类之间，选**最邻近距离最小**的一对合并；结果是树状图（dendrogram），横切一刀得到任意类数。

**优缺点**：
- ✅ 得到类别间**层次联系**，反映不同相似水平上的聚类情况
- ✅ **不用预先指定类别数**（事后在树上选）
- ❌ **计算和空间复杂度很高**（每步可能要判断所有样本对）

> 📜 **侧栏一句**：课件在"聚类算法"总览里还点名了概率分布聚类（混合高斯）和密度聚类（**DBSCAN**、OPTICS——把类定义为高密度区域），但未展开。论述题里作为"还有哪些流派"的一句话点缀即可，不必展开原理。

**K-means vs 层次聚类 对比表**（章内对比题预制件）：

| | K-means | 层次聚类 |
|---|---|---|
| 预先给 k？ | **要** | 不要 |
| 输出 | 一个划分 | 层次树（多粒度划分） |
| 复杂度 | 低 | **很高** |
| 对离群点 | 非常敏感 | 相对不那么致命 |
| 适用形态 | 成团分布 | 更灵活 |

## 3.6 聚类的四条"性格"（课件原文，简答好料）

1. **无需先验知识**（类别内部特征、合理数量都不用事先知道）
2. 结果只是划分了相似样本，**类别的具体含义要靠后续分析确定**
3. **加入新数据后原聚类结果会改变**（不稳定）
4. 有助于对样本组织后**高效检索**

## 3.7 分类：定义与过程

**分类（Classification）**：基于**已知样本类别集合**，学习决策规则，对未知类别样本进行类别划分。
两阶段：**训练（学习）**——用已知类别样本求解模型参数；**分类（预测）**——用训练好的模型给新样本定类。

**分类基本过程**（六步）：
1. 确定感兴趣的类别 → 2. 建立本质属性指标体系 → 3. 选择/标注**训练样本** → 4. 训练分类模型 → 5. **评估分类精度** → 6. 对新样本分类

## 3.8 决策树 ★本讲唯一展开讲的分类模型

**结构**：树形流程式模型；**分支节点** = 对某一样本特征的划分；**叶节点** = 判断出的类别；一般为**二叉树**；决策过程可**显式表示**。

**构建三步骤**（算法有 ID3、C4.5、CART，认得名字即可）：
1. **特征选择**：选一个特征作当前节点分裂标准
2. **新增节点**：按划分准则分裂，重复至无法生长。划分准则 = 每次划分使子集**纯度更高**（如**信息熵减少**）
3. **剪枝**：决策树容易过度划分特征空间，需去除一些分支

几何理解：生成一个节点 = 把特征空间**切一刀**。

**优缺点**（必背四条）：
- ✅ 参数和分类过程**易于解释**，有助于分析类间差异
- ✅ **不要求样本满足统计分布假设**（如正态性）
- ❌ 对训练样本**敏感性很高**：样本微小变动 → 模型和精度大变
- ❌ **非常容易过拟合**（训练集准、新样本差），需结合剪枝

## 3.9 分类模型评估：混淆矩阵 → 精度 → Kappa ★必考

评估前先把已知样本分成**训练集**（训模型）和**测试集**（评估）。

**混淆矩阵**：行 = 模型分类结果，列 = 真实类别，交叉计数。
**总体分类精度** = 对角线（正确分类数）之和 ÷ 测试样本总数。课件例：(239+73+280)/664 = 89.2%。

**总体精度的坑**（一定考）：无法反映**类别不平衡**带来的虚高。若 99% 样本是"存活"，模型无脑全判"存活"也有 99% 精度，但对识别病情毫无用处。

**Kappa 系数**：K = (P₀ − Pₑ)/(1 − Pₑ)，其中 P₀ = 总体精度，Pₑ = **偶然一致性**比例（按行列边际算"瞎猜也能蒙对"的期望）。
**读数**：区间 [−1, 1]；**接近 1** 分类与真实高度一致；**接近 0** 结果可能纯属偶然；**接近 −1** 严重不一致。
一句话本质：**从精度里扣掉"运气分"**。

## 3.10 监督 vs 非监督分类 ★对比必考

| | 监督分类 | 非监督分类 |
|---|---|---|
| 先验知识 | **需要**：部分样本的类别标签 + 类别先验知识 | **不施加任何先验**，只凭数据特征分布（自然聚类特性） |
| 学习方式 | **先学习后分类** | **边学习边分类** |
| 前置工作 | 分类前须收集/圈定**性质单一的训练集** | 不需要 |
| 结果含义 | 类别有明确语义（就是标注的那些类） | 只做到"区分"，**类别实际含义需后续分析确定** |
| 代表 | 决策树、朴素贝叶斯 | 聚类（一般认为非监督分类就是聚类） |
| 比喻 | 有老师批改作业 | 自习自己悟 |

**聚类与分类的三条区别**（课件原文，简答直接背）：
1. 分类把样本划入**已有**类别；聚类把相似样本归为一类
2. 分类**需要**类别先验知识；聚类**不依赖**先验
3. 聚类一般用**无标注**样本；分类主要用**有标注**样本

## 易错点

1. "聚类可视为分类的一种"（课件原话）但**非监督分类≈聚类**——两句话别搅在一起：广义上都叫"给样本定类"，狭义方法论上监督=分类、非监督=聚类。
2. Kappa 接近 **0** 是"偶然"不是"最差"；最差是 −1。
3. DBI **越小越好**（不是越大）。
4. K-means 对离群点**敏感**，决策树对训练样本**敏感**——两个"敏感"对象不同。
5. 决策树不要求正态性——别把第 2 讲 t 检验的前提搬过来。
6. 过拟合定义：**训练样本上准、其他样本上差**——两半都要写全。

**本讲在全课程主线中的位置**：本讲处理的是"属性空间"里的相似性（还没用地理位置）；第 4 讲开始，"相似性"加上空间坐标，升级为空间自相关——"特征空间的近"变成"地理空间的近"。

# 第 4 讲 相关性与空间自相关 ★全卷最重

## 【1分钟速览】

上半场**相关性**：Pearson（线性、要正态、怕极端值）→ Spearman（秩、抗极端值、能测单调非线性）→ Kendall（一致对/不一致对）→ 偏相关（控制第三变量）。下半场**空间自相关**：先定义空间邻近（邻接：Rook/Bishop/Queen；距离：欧氏/球面）→ **全局**指标（Moran's I、Geary's C、Getis-Ord G）判断整个区域有没有聚集 → **局部**指标（局部 Moran's I、Gi\*）找出聚在**哪儿**（热点/冷点）。所有指标都要做 **Z 检验**定显著性。

## 4.1 相关性的基本概念

**相关分析**：研究现象之间的**依存关系**，探讨其相关**方向**与相关**程度**的统计方法。

三组必辨概念：
- **相关关系 vs 确定关系**：确定关系可用数学公式准确表达（正方形面积-边长）；相关关系是**非完全确定的趋势**，不能用公式准确表达——相关关系才是相关分析的研究对象。
- **相关 ≠ 因果**：可能仅是关联。课件例：两个同年出生的人身高逐年共同增长，但无因果。
- 分析前先画**散点图**：判断线性/非线性/无相关、相关程度、异常点影响——"判断是否线性相关是相关分析的重要准备"。

多变量情形的工具：**偏相关系数、复相关系数、典型相关分析**（后两个认得名字即可）。

## 4.2 三大相关系数 ★对比必考

### Pearson 积差相关系数

对 n 对观测 (x, y)：分子是 x、y 对各自均值的**离差乘积和**，分母是两者离差平方和的乘积开根号。样本相关系数记 **r**，总体记 ρ。

**读数**：值域 [−1, 1]；r>0 正相关，r<0 负相关，**r=0 只说明无明显线性关系**（可能有很强的非线性关系！）。

**显著性检验**（为什么要检验：r 接近 1 的程度与样本量有关——**n=2 时 r 恒等于 1**，小样本的 r 波动大）：
H₀: ρ=0；统计量 t = r/√((1−r²)/(n−2)) ~ t(n−2)。|t| 超临界值 → 相关显著。

**适用条件**（必背三条）：① 两变量为**间隔或比例水平**数据；② 两变量**符合正态分布**；③ 只能测**线性**相关。
另两个坑：**易受极端值影响**；相关系数**不是等距测量**——r 从 0.3 变 0.6 不代表相关程度翻倍。

### Spearman 等级相关系数

**等级（秩，rank）**：样本按大小（优劣）排序的位次。Spearman 用**秩**而不是原始值计算。

公式（这条建议记形）：**r_s = 1 − 6Σd²/(n(n²−1))**，d 为每对样本的**等级之差**，n 为样本量。读法：秩完全一致 → Σd²=0 → r_s=1；秩完全颠倒 → r_s=−1。

显著性：小样本查 Spearman 检验表；**n>20** 时可用 t = r_s√((n−2)/(1−r_s²)) ~ t(n−2)。

**适用条件**：① 变量可为**连续或离散**数据；② **无需正态分布**；③ 反映**单调关系**强弱，因此**能衡量非线性相关**（任何单调曲线秩相关都是 1）；④ **受极端值影响很小**。近似线性时 Spearman ≈ Pearson。

### Kendall 等级相关系数 τ

也是秩相关，但思路不同：数**样本对**。对两个样本对，若两个变量上的大小方向一致 → **一致对**；方向相反 → **不一致对**。
**τ = (一致对数 − 不一致对数) / 总对数**，总对数 = n(n−1)/2。读法：全体样本对都"同向"则 τ=1。

### 三系数对比表（背这张）

| | Pearson | Spearman | Kendall |
|---|---|---|---|
| 用什么算 | 原始观测值 | 秩（等级） | 样本对的一致性 |
| 数据要求 | 间隔/比例水平 | 次序水平即可（连续或离散） | 次序水平即可 |
| 正态要求 | **要** | 不要 | 不要 |
| 测什么关系 | **线性** | 单调（含非线性） | 单调（含非线性） |
| 极端值 | **敏感** | 影响很小 | 影响很小 |

### 偏相关分析

**动机**：一个结果常由多因素综合作用产生。作物产量既与降雨有关也受温度影响——直接算产量-降雨相关，混入了温度的作用。**偏相关 = 控制（剔除）其他变量影响后，两个变量的直线相关程度**，更能反映"真实关系"。

按固定变量个数分阶：**零阶偏相关 = 普通相关**；固定 1 个 → 一阶；固定 p−1 个 → p−1 阶。一阶公式结构：r_xy·z = (r_xy − r_xz·r_yz)/√((1−r_xz²)(1−r_yz²))——分子是"总相关减去经由 z 的间接相关"。值域仍 [−1,1]，绝对值越大偏相关越强。

## 4.3 空间自相关：概念与空间邻近

**空间自相关（Spatial Autocorrelation）**：一些变量在**同一分布区内观测数据之间**潜在的相互依赖性。理论依据即 Tobler 第一定律。三种模式：空间邻近的属性相似（**正**自相关/聚集）｜相互独立（**随机**）｜邻近属性不相似（**负**自相关/棋盘格）。

再精确化一对定义（和第 1 讲呼应，本讲课件重新给了定义）：
- **空间依赖性**：一个区域某地理现象属性值，与**邻近区域同一现象**属性值的相关程度。
- **空间异质性**：每个空间单元上的事物现象都有别于其他区位，反映**空间非平稳性**。

**空间邻近（Neighborhood）的两种描述** ★名词解释高危：

**① 邻接关系**（公共边界非零长度；名义的、双向的、相等的距离）——国际象棋记忆法：

| 类型 | 判邻标准 | 棋子记忆 |
|------|---------|---------|
| **Rook（车）** | 有**共同边界**（边相邻） | 车走直线→共享"边" |
| **Bishop（象）** | 有**公共顶点**（点相邻） | 象走斜线→只碰"角" |
| **Queen（后）** | 共同边界**或**相同顶点 | 后=车+象，通吃 |

**② 距离关系**：**欧氏距离**（平面直线）与**球面距离**（大圆距离，经纬度+地球半径反余弦公式，认得即可）。

## 4.4 全局 vs 局部空间自相关 ★简答高频

空间自相关统计量反映**空间依赖性**程度，也一定程度反映空间异质性。分两级：

| | 全局指标 | 局部指标 |
|---|---|---|
| 问题 | 整个区域**有没有**聚集？ | 聚集/突变**在哪儿**？ |
| 输出 | **单一值**概括全区 | **每个空间单元一个值** |
| 探测 | 整体空间分布模式 | 属性变化是平滑（均质）还是突变（异质）；热点/高发区域 |
| 成员 | Moran's I、Geary's C、Getis-Ord G | 局部 Moran's I、局部 Geary's C、Getis-Ord **Gi\*** |
| 局限 | 掩盖局部差异 | 弥补全局分析的不足 |

## 4.5 全局指标

### 全局 Moran's I ★必考之王

**思想**：比较邻近空间位置的观测值**与均值偏差的乘积**。
公式骨架：I = [ΣΣ w_ij (x_i − x̄)(x_j − x̄)] / (σ² ΣΣ w_ij)。
**逐符号读**：w_ij = 单元 i、j 的**邻近性度量**（邻接 0/1 或距离权重）；分子 = 所有邻近对的离差乘积加权和（两个都高于均值或都低于均值 → 正贡献）；除以方差和权重总和做归一。**结构上就是"Pearson 相关系数的空间版"——把'另一个变量'换成'邻居的自己'。**

**读数**（课件原话）：值域一般 [−1, 1]（计算前常对观测值归一化）
- **I > 0**：正自相关——相似值（同高或同低）的单元聚集
- **I = 0**：不相关——可能是随机分布
- **I < 0**：负自相关——相异值（高低相间）的单元聚集

**显著性检验**：构造 Z = (I − E(I))/√Var(I)，其中 **E(I) = −1/(n−1)**（随机情形的期望，注意不是 0！）。
- Z ≥ Z_α：相似观测值单元倾向聚集
- Z ≤ −Z_α：不同观测值单元倾向聚集
- −Z_α < Z < Z_α：接受 H₀（不存在空间自相关），不显著

### Getis-Ord G（全局）

**思想**：设定一个**距离阈值 d**，在阈值内定义空间关系，直接用**邻近位置观测值的乘积**（不减均值！）来测相似程度。G(d) = ΣΣ w_ij(d)·x_i·x_j / ΣΣ x_i·x_j。
显著性同样走 Z 得分检验：
- **Z 为正**：观测值**较高**的对象存在空间聚集（高值扎堆）
- **Z 为负**：观测值**较低**的对象存在空间聚集（低值扎堆）

**Moran's I vs G 的分工**（对比题金句）：Moran's I 只能说"有聚集"，但**分不清是高值聚集还是低值聚集**（高高聚和低低聚都给正 I）；G 统计量能**区分高值聚集与低值聚集**。

> 📜 **侧栏一句**：Geary's C 在全局指标名单里被点名（第 5 讲还用它诊断残差：C ≥ 0.2 说明残差明显高低聚集），但公式未展开——记住它与 Moran's I 互补、基于"差值平方"而非"离差乘积"即可。

## 4.6 局部指标

### 局部 Moran's I（LISA）

对**第 i 个单元**：I_i = (x_i − x̄)/σ² · Σ_j w_ij (x_j − x̄)——读法：**我的离差 × 我邻居们的加权离差和**。同样有 Z_i 检验。I_i 显著为正 = 我和邻居同高（高高）或同低（低低）；显著为负 = 我和邻居唱反调（高低/低高，空间离群点）。

### Getis-Ord Gi\* ★热点冷点分析

**能区分"热点"和"冷点"**：高-高局部聚集 = 热点，低-低局部聚集 = 冷点。与全局 G 一样用**空间距离**定义邻近。检验统计量 Z(Gi\*)：
- **Z > 0 且显著 → 热点区**（高值聚集）
- **Z < 0 且显著 → 冷点区**（低值聚集）

**局部 Moran vs Gi\* 一句话分工**：局部 Moran 能同时揪出**空间离群点**（高低/低高），Gi\* 专职把**热区冷区**分级画出来。

## 4.7 深圳案例（附加题/论述题素材）

深圳市**交通分析小区（TAZ）**尺度、某工作日人口密度为观测值，邻近性取**欧氏距离倒数**：
- 全局 Moran's I 为正、显著性检验 **p 值接近 0** → 拒绝 H₀ → 高人口密度 TAZ 空间聚集，低密度 TAZ 也聚集（同城多中心格局的定量证据）
- 再算局部 Moran's I 与 Z 值图 → 用 **Gi\*** 输出不同置信度（α=0.1/0.05/0.01）的热点/冷点区域图

**这就是标准作业流**（论述题万能模板）：全局指标定性"有没有聚集" → 局部指标定位"聚在哪" → 分置信度出热冷点图 → 解释与决策。

## 易错点

1. **r=0 ≠ 无关系**，只是无**线性**关系（Spearman 可能照样等于 1）。
2. Pearson 要正态、Spearman/Kendall 不要——正态要求只属于 Pearson。
3. Rook=共**边**，Bishop=共**顶点**，Queen=边或顶点——别把车和象记反（象走斜线只沾角）。
4. Moran's I 的随机期望是 **−1/(n−1)**，不是 0。
5. Moran's I **负值 ≠ 不相关**：负值是"高低相间"的负自相关；接近 E(I) 才是随机。
6. I 分不清高聚还是低聚；要分高低用 **G / Gi\***。
7. 局部指标每个单元**一个值**；全局全区**一个值**——答题时"单一数值概括"这几个字要写出来。

**本讲在全课程主线中的位置**：把第 2 讲的"相关"装上空间坐标，是全课程的枢纽——它诊断出的空间依赖，正是第 5 讲空间回归要"治"的病、第 7 讲插值要"用"的药。

# 第 5 讲 回归分析与空间回归

## 【1分钟速览】

三级火箭：**经典回归**（OLS 最小二乘，SST=SSR+SSE，R² 拟合优度，F 检验方程、t 检验系数）→ 残差若有空间自相关，经典假设（独立性）被打破 → **空间自回归 SAR**（因变量带空间滞后 → **SLM**；误差项带空间相关 → **SEM**；再加邻居自变量 → **杜宾**；用 **LM 检验**四件套选型）→ 系数若随地点漂移（空间异质性）→ **GWR**（每个位置一套局部系数，WLS+空间权函数+带宽）。

## 5.1 回归的由来与定位

> 📜 **历史锚点**："回归"一词来自**高尔顿（Galton）与皮尔逊**的遗传学研究：高个子父母的子女平均身高**低于**父代，矮个子的**高于**父代——后代身高有"**向平均回归**"的趋势，离均值越远回归压力越大。这个词本义是"退回均值"，后来泛化为一切"用 X 解释 Y"的建模。记住这个故事，"regression"就再也不是黑话。

**为什么相关分析不够**（简答直给）：相关分析 ① **没有模型/公式**描述变量关系；② **不能对未知样本预测**。回归分析补上这两块。

**回归分析定义**：对存在相关的变量，取一个/多个作**自变量**、另一个作**因变量**，用**数学方程式**表达其相关关系，从而由自变量**估计、预测**因变量。方程称回归模型。

分类：按变量个数——**一元/多元**；按关系形态——**线性/非线性**。

**回归 vs 相关 三条区别**（★经典简答）：
1. 相关分析中两个变量**都是随机变量**；回归分析区分自/因变量，因变量随机、**自变量假定为非随机可控变量**
2. 相关系数只反映**密切程度**，给不出数量变化关系；回归建立**方程**，可定量揭示影响并**预测**
3. 两者互为基础：相关分析需回归表明数量关系的具体形式，回归应**建立在相关分析基础上**（先确认相关再回归）

回归分析要回答的四个基本问题：变量是否相关？模型怎么求？拟合效果怎么评？预测结果怎么评？——本讲结构就是按这四问展开的。

## 5.2 一元线性回归

**模型**：y = β₀ + β₁x + ε（ε 为随机误差项，无法用 x 解释的部分）。拟合方程记 ŷ = b₀ + b₁x（b 是 β 的估计，ŷ 是拟合/预测值）。

**基本假设**（必背三条）：
1. x 与 y 具有**线性关系**（最基本前提）
2. 对每个 x 值，对应的 y 子总体服从**正态分布**
3. 不同 x 对应的 y 组之间**相互独立**  ← 空间数据违反的就是这条！

**系数几何意义**：b₀ = 截距（x=0 时 y 的估计值；>0 交纵轴于原点上方）；**b₁ = 斜率 = x 每变化一个单位，y 的平均改变量**。

**参数求解：最小二乘法（LSM）**：以**残差**（观测值 − 拟合值）的**平方和 Q** 为标准（平方消符号），找使 Q 最小的 b₀、b₁——求极值问题，对两参数求偏导置零解出。也可用**最大似然估计（ML）**。

## 5.3 回归模型检验 ★简答高频区

**平方和分解**（必背）：**SST = SSR + SSE**

| 项 | 名称 | 含义 | 自由度 |
|----|------|------|--------|
| SST | 总平方和 | 因变量 y 的总变动 | n−1 |
| SSR | 回归平方和 | 回归方程**解释到**的变动，**越大拟合越好** | 1 |
| SSE | 残差平方和 | 回归方程**没解释**的变动，**越大拟合越差** | n−2 |

**三种检验**（考"如何评价回归模型"就答这三层）：

1. **拟合优度：决定系数 R² = SSR/SST**——自变量能解释因变量变动的**比率**；取值 [0,1]，越接近 1 越好。课件例：R²=0.796 → 土壤侵蚀变化可由降雨变化解释 79.6%（且斜率 0.9269：降雨每 +1mm，侵蚀量 +0.9269）。
2. **方程显著性：F 检验**——R² 没考虑样本量，不算严格检验。F = SSR/(SSE/(n−2)) ~ F(1, n−2)；H₀: β₁=0（无线性关系）。F 超临界值 → 回归方程显著。称为**一元回归的方差分析**。
3. **系数显著性：t 检验**——直接检验 β₁ 是否显著≠0。一元下结论与 F 检验相同；**多元回归里 t 检验才显威力**：对每个自变量的系数单独检验，判断该自变量是否有显著影响。

## 5.4 多元线性回归与非线性回归

**多元模型**：y = β₀ + Σβₖxₖ + ε。βₖ 称**偏回归系数**：**其他自变量固定时**，xₖ 每变一单位 y 的平均改变量。回归函数称**回归曲面（响应曲面）**。

**经典多元回归基本假设**（四条，必背——空间回归就是冲着拆它来的）：
1. 自变量之间相互独立（**无多重共线性**）
2. 随机扰动项零期望、**同方差**、序列不相关
3. 自变量与随机扰动项**不相关**
4. 随机扰动项服从均值为零的**正态分布**

参数估计：普通最小二乘（OLS）、极大似然、矩估计。

**非线性回归**两类：**内线性模型**——参数非线性但**可变换化为线性**（y=β₀e^(β₁x)ε 取对数即线性）；**狭义非线性模型**——变换也救不回来的。

## 5.5 空间自回归（SAR）★论述重灾区

### 为什么需要

- 空间依赖 = 空间观测值**不独立** → 经典回归的独立性等严格假设**被违反**，无法把空间相关性考虑进去。
- 直观例（课件）：东京某地劳动年龄死亡人数上升会"传染"到周边（未观测的空间相关因素如辐射）；意大利新冠患病率影响法国英国，却影响不了地理遥远的韩国。
- **诊断**：对经典回归的**残差**算全局 Moran's I / Geary's C。**残差的高值/低值在空间上显著聚集 = 模型缺了某个关键（空间）变量**（Geary's C ≥ 0.2 一般说明残差有明显高低聚集）。

### 核心构件：空间滞后与空间权重矩阵 W

**空间滞后**：空间依赖由邻域向邻域扩散的空间效应；反映**相邻空间单元因变量取值的相似性**。数学表达 y\* = ρWy。
**空间权重矩阵 W（空间滞后算子）**：n×n 矩阵，w_ij 描述单元 i、j 的邻近程度（**邻接关系或距离权重**——就是第 4 讲那套 Rook/Queen/距离），对角线为 0。Wy 的含义：**每个单元的邻居因变量的（距离加权）平均值**——"空间上的推移"。

### 一般形式与家族谱系 ★必背

**一般形式**：y = ρW₁y + Xβ + ε，  ε = λW₂ε + μ
逐符号：ρ = **空间滞后系数**（因变量的空间趋势）；λ = **空间误差系数**（残差的空间趋势）；W₁/W₂ = 相应的空间权重矩阵；μ = 独立正态噪声。

| ρ | λ | 模型 |
|---|---|------|
| 0 | 0 | 普通线性回归（OLS） |
| ≠0 | 0 | **空间滞后模型 SLM**（无 Xβ 时为一阶空间自回归） |
| 0 | ≠0 | **空间误差模型 SEM** |
| ≠0，且加 WXδ 项 | — | **空间杜宾模型 SDM** |

**SLM（空间滞后模型/混合回归与空间自回归模型）**：y = ρWy + Xβ + ε。
认为**相邻区域间的因变量存在空间自相关**：我的 y 与我的 X 有关，也与**邻居的 y** 有关。ρ∈[0,1]，越接近 1 相邻地区因变量越相似。适用直觉：**扩散/溢出过程**（传染病、房价传导——邻居的结果直接影响我的结果）。

**SEM（空间误差模型/空间残差自回归模型）**：y = Xβ + ε，ε = λWε + μ。
**因变量本身无空间自相关，但误差（未观测因素）空间相关**：我的 y 受我的 X 和**邻居的误差**影响。λ∈[0,1]，越接近 1 相邻地区（误差反映的遗漏变量）越相似。适用直觉：**遗漏了某个本身空间聚集的变量**（如空气质量、政策区划）。

**SDM（空间杜宾模型）**：y = ρWy + Xβ + WXδ + ε。
我的 y 与我的 X、**邻居的 y、邻居的 X** 都有关；δ 度量**相邻区域自变量对本地因变量的边际影响**。SLM 的加强版。

**SLM vs SEM 一句话本质**（易混王牌）：**SLM 说"邻居的结果影响我"，SEM 说"邻居和我共享看不见的原因"。**

### 估计与选型

- 三种空间模型**不能用 OLS** 估计参数 → 用**最大似然（ML）或广义矩估计（GMM）**。（为什么：滞后项 Wy 和误差同时出现，OLS 估计有偏——课件只要求记结论。）
- **LM（拉格朗日乘数）检验**四件套（服从卡方分布）：**LM-error / Robust LM-error**（检验残差空间相关性）+ **LM-lag / Robust LM-lag**（检验因变量空间自回归效应）。
- **选型流程**（论述题可画流程图）：① 先跑 **OLS**，得残差；② 算 LM 统计量；③ LM-lag 显著 & LM-error 不显著 → SLM；反之 → SEM；都显著 → 看 **Robust** 版本谁显著用谁。
- 课件案例（美国南部县室内极端案件 ~ 十项社会经济变量）：OLS R²=0.104，LM 四项中 **Robust LM-lag 显著、Robust LM-error 不显著 → 选 SLM**，R² 升至 0.159，残差 Moran's I 降到 −0.022（≈无残余空间自相关）。**这个"R² 提升 + 残差自相关消失"的双证据结构，是论述题的标准收尾。**

## 5.6 地理加权回归（GWR）★论述重灾区之二

### 动机：空间非平稳性

**空间非平稳性**：因地理位置变化而引起的**变量间关系或结构的变化**（各种事物空间上缺乏平稳结构；空间单元本身不均质）。
全局模型（OLS、**SLM/SEM/SDM 也都是全局模型**——系数全区一套）事先假定关系同质，结果只是全区的某种"**平均**"，**掩盖了变量关系的局部特性**。

### 模型

**GWR = 把数据的地理位置嵌入回归参数**：
y_i = β₀(u_i, v_i) + Σₖ βₖ(u_i, v_i)·x_ik + ε_i
(u_i, v_i) 是第 i 个采样点坐标；**每个回归参数都是地理位置的函数**——每个地点一套系数。若所有位置的 βₖ 全相等，GWR 退化为普通线性回归。

### 参数估计：WLS + 空间权函数 + 带宽

问题：每点一套参数 → 未知数远多于观测数。解法：假设**回归参数是连续表面，邻近位置参数相似** → 用**该位置邻近的观测组成局部子集**，做**加权最小二乘（WLS）**——离回归点越近的观测权重越大。权重构成对角阵（又见空间权重矩阵！课件自问："还有哪些指标用了空间权重矩阵？"——Moran's I、SLM/SEM、GWR，跨章题就考这个）。

**空间权函数**（权随距离递减）：全局函数（w=1，退化为 OLS）、**距离阈值函数 Box-Car**（阈内 1 阈外 0）、**指数型**、**高斯型**、**双重平方 Bi-square**、三次立方（认得名字+"都是距离衰减函数"即可）。

**带宽 b**（★必考概念）：描述**权重随距离衰减快慢**的非负参数，越大衰减越慢。
- 带宽**过大** → 参数估计**偏差**过大（局部被抹平，趋向全局平均）
- 带宽**过小** → 参数估计**方差**过大（局部样本太少，估计不稳）
——经典**偏差-方差权衡**。选带宽可用交叉验证（CV）或 **AIC 准则**（案例里都出现过）。

**固定带宽 vs 自适应带宽**：固定最优带宽适合**空间单元均匀分布**；分布不均时某些回归点附近没数据 → 方差爆炸 → 用**自适应空间权函数**（按数据疏密调带宽，如取最邻近 k 个单元）。

### 检验与扩展

两类假设检验（认得即可）：① **模型的空间非平稳性检验**——比较 GWR 与 OLS 的残差平方和（F 检验），看 GWR 是否显著更优；② **回归参数的空间非平稳性检验**——看某系数在空间上的变异是否显著。

**混合 GWR**：并非所有参数都随地点变——一部分**变参数**（随位置变）+ 一部分**常参数**（全局固定）混合；去掉常参数部分回到 GWR，去掉变参数部分回到普通回归。

### 两个案例（论述题素材）

1. **美国膀胱癌死亡率**（y=膀胱癌死亡率，x=肺癌死亡率、人口密度对数，506 个经济区）：OLS R²=0.25 → GWR（高斯权函数，CV 选带宽 1.27）**R²=0.52，翻倍**；且系数空间分布揭示部分地区出现**违反直觉的负相关**——全局回归里被"平均"掉的局部故事。
2. **佐治亚州学士学位比例**（人口普查变量拟合）：OLS R²=0.24 → GWR（高斯核+AIC 选带宽 49.46）R²=0.30；农村人口、贫困、非裔比例的系数各有空间格局。**金句：GWR 能更大限度地挖掘和展示空间异质性。**

## 易错点

1. **SLM vs SEM**：滞后模型是**ρWy**（邻居的因变量进方程）；误差模型是**λWε**（空间相关藏在误差里）。判词：结果传染 → SLM；共享隐因 → SEM。
2. **SAR 家族治"空间依赖"，GWR 治"空间异质性"**——病不同，药不同；SLM/SEM/SDM 是**全局**模型（系数一套），GWR 是**局部**模型（每点一套）。
3. R² = SSR/SST（不是 SSE/SST）；SSE 大 = 拟合差。
4. F 检验查方程整体，t 检验查单个系数——多元回归里两者不再等价。
5. 空间回归参数**不能 OLS**，要 ML/GMM。
6. 带宽大 → **偏差**大；带宽小 → **方差**大——两个词别写反。
7. 偏回归系数含义必须带"**其他自变量固定时**"这个前置条件。
8. 回归假定**自变量非随机**、因变量随机——与相关分析（都随机）的第一区别。

**本讲在全课程主线中的位置**：本课程的"顶点"——第 2 讲的回归 + 第 4 讲的空间自相关诊断，合成两条修复路线：依赖 → SAR 家族，异质 → GWR；第 1 讲埋的"依赖/异质"二分在这里收线。

# 第 6 讲 网络分析

> ⚠️ 提醒：本讲课件实际结构是 **邻近度分析 → 叠加分析 → 网络分析 → 可达性分析** 四块（不含"网络中心性"专题）。

## 【1分钟速览】

先扩展"距离"概念（欧氏/曼哈顿/**网络距离**）。**邻近度分析**两件套：缓冲区（辐射范围）、泰森多边形（最近邻分区）。**叠加分析**：矢量（相交/联合/标识等）与栅格（土地适宜性评价）。**网络分析**：网络 G=<N,E,W>（节点/边/权重），存储用邻接矩阵/关联矩阵/权重表；四大功能——路径分析（**Dijkstra 最短路径**）、连通分析（最小生成树）、定位与配置（选址/分配）、网络流。**可达性分析**三大方法：空间阻隔法（距离越小越可达）、机会累积法（阈值内机会越多越可达）、空间相互作用法（Shen 供需势能模型）。

## 6.1 距离的再认识

**距离的定义与度量空间有关**——欧氏距离只适用于**均质化**的度量空间。三种距离：

| 距离 | 定义 | 适用 |
|------|------|------|
| 欧氏距离 | 两点直线 | 均质空间 |
| 曼哈顿距离 | \|Δx\| + \|Δy\|（横平竖直走） | 格网街区 |
| **网络距离** | 沿网络路径的长度和 | 真实交通/管网 |

（不同数据形态还有点-线距离、面-面距离、向量余弦距离等。）**本讲的存在理由**：城市里两点的"近"，不是直线近，而是**路网上近**——这是网络分析对前几讲"距离"的升级。

## 6.2 邻近度分析

**邻近度（proximity）**：空间实体距离相近的程度。

### 缓冲区（Buffer）

**定义**：根据点、线、面实体，自动建立其周围**一定距离的带状区**，用于识别实体对邻近对象的**辐射范围或影响程度**。集合语言：给定对象 O，邻域 = 距 O 不超过半径 R 的点集。
应用例：道路两侧安全带、公交站 800m 步行服务范围、道路维修 60m 内受噪声影响的建筑（主体=道路，邻近对象=建筑物，作用条件=噪声强度随距离变化）。
形式：矢量缓冲区（点/线/面 × 单要素/多要素/分级要素）；栅格缓冲区（四邻域/八邻域扩张）。

### 泰森多边形（Thiessen Polygon）

> 📜 **历史锚点**：荷兰气象学家 **A. H. Thiessen** 提出，原用于**由离散雨量站估计降水量分布**——它天生就是插值工具，所以第 7 讲还会再见到它。

**定义特点**（必背两条）：① 每个多边形区域内任意位置**到本区内采样点的距离小于到其他任何采样点的距离**；② 区内位置的目标值可由该区采样点决定——故又称**最邻近点法**。
**生成方法**：采样点连成 **Delaunay 三角网** → 作各边**垂直平分线** → 围成泰森多边形（= Voronoi 图）。（另一路：区域栅格化、由采样点向外扩张再矢量化。）
应用例：每栋建筑归属最邻近通信基站。

## 6.3 叠加分析

**空间叠置分析**：统一空间参照系下，将同一地区**两个图层**叠置，产生**空间区域的多重属性特征**或建立对象间空间对应关系。例：水污染区多边形 ∩ 建筑物图层 → 受污染建筑。

- **矢量叠加**：图层叠加后**求交、拓扑生成和属性处理**（空间逻辑运算）；操作类型认名字：相交、联合、标识、更新、交集取反。
- **栅格叠加**：典型问题**土地适宜性评价**——土壤深度/类型、坡度、DEM 等栅格 → 重分类 → 叠置函数计算（含水量、侵蚀度等）→ 综合得土地质量与适宜性。**逐像元的"多图层代数运算"**。

## 6.4 网络分析

**网络（network）**：用于实现**资源运输和信息交流**的一系列**相互连接的线性特征**组合。图的基本表达 **G = <N, E, W>**：**节点 node**（线段交点：港口、车站）、**边 edge**（弧段：公路、河流）、**权重**（边上速度/长度，节点上库存；带权的图叫赋权图）。

**拓扑关系**：边或节点之间的**连接关系**，不受具体空间坐标、旋转和拉伸影响。矢量数据一般没有拓扑结构，网络分析时可产生**临时拓扑**。

**存储方法**（三个名词认得）：**邻接矩阵**（节点×节点，谁挨谁）、**关联矩阵**（节点×边，谁连哪条边）、**权重表**。

**网络分析定义**：对地理网络、城市基础设施网络进行**地理化和模型化**，研究网络状态、模拟资源在网络上的**流动和分配**，实现网络结构与资源的**优化**。理论工具：**图论 + 运筹学**。

**四大基本功能**（必背表）：

| 功能 | 内容 |
|------|------|
| **路径分析** | 指定条件下的资源流动最优路径：最短路径、最快路径、邮递员问题 |
| **连通分析** | 连通分量求解；最小费用连通方案 |
| **定位与资源配置** | 定位问题（已知需求源→布设供应点=**选址**）；分配问题（需求源接受哪个供应点服务） |
| **网络流分析** | 物资从产地经运输网络到市场的运输问题 |

### 路径分析与 Dijkstra 算法 ★本讲核心

两类典型问题：**最短路径问题**（给定起终点，求时间/成本最小路径）；**旅行商问题 TSP**（从起点出发经所有节点恰一次回到起点，总距离最小）。

**Dijkstra（1959）**：解决**单源最短路径**（一个起点到其余所有节点），**按路径长度递增的次序**逐个产生最短路径。
**流程**（能口述）：
1. 初始化：起点距离标记 0，其余全 ∞
2. 维护"已定集合 S / 未定集合 T"；每轮从 T 中选**当前距离标记最小**的节点进 S
3. **核心步骤（松弛）**：用刚进 S 的节点 u 更新它的邻居 v：d(v) = min{d(v), d(u)+w(u,v)}
4. 重复直到 T 空；若要路径本身，更新时**记录前驱节点**，最后**回溯**

**Dijkstra 特点**（必背四条）：① 处理**单一起点**问题；② 可求解**有向图**；③ **不能处理负权重**；④ 效率较低，节点增多耗时急剧上升。
**替补队员**：**Floyd** 算法——求**两两节点间**最短路径，可处理负权；**Bellman-Ford**——单源但**可处理负权**。

### 连通分析

两节点间存在路径即**连通**。典型问题：① **连通分量求解**——河网上某水坝拉闸，找受影响的下游节点（**广度/深度优先搜索**）；② **最小费用连通**——6 城市修铁路全连通且总长最小 = **最小生成树问题（Kruskal 算法）**。

### 定位与配置分析

目标：选定供应中心并把边/点分配给中心，使各服务区内**总加权距离最小**。
**网络服务区 ≠ 缓冲区**（易考对比）：服务区按**网络路径**产生边界；缓冲区按**直线距离**产生边界——形状差异明显。
- **定位（选址）**例：学校选址——道路网以出行时间为权，找四个居民点 10 分钟最短路径的**重合节点**附近建校。
- **分配**例：消防站——以道路距离为权，从各消防中心**同时广度搜索**，直到各中心分足 10 条街道（每街 10 户、每中心容量 100 户）。

## 6.5 可达性分析 ★简答/论述好出

**可达性（accessibility)**：反映**交通成本**的基本指标，度量空间上某一实体**位置的优劣程度**。无统一定义，三种主流观点：**克服空间阻隔的难易**｜**单位时间内可接近的发展机会数量**｜**相互作用的潜力**。

### 三大方法对比（背这张表）

| 方法 | 思想 | 优点 | 缺点 |
|------|------|------|------|
| **空间阻隔法**（距离法） | 阻隔越低可达性越高；指标：直线距离/网络距离/出行费用 | 计算简单、直观易懂 | **忽略目的地吸引力与人的活动因素** |
| **机会累积法** | 给定时间/距离/成本**阈值**内能接近的**目标点数量**，越多越可达 | 与习惯思维一致；直观显示距离-机会关系；同范围内可比 | 出行范围取得足够大人人满分，**无法反映区域体系中的地位**；阈值不同结果变，不利客观比较 |
| **空间相互作用法** | 从供需相互作用角度评价；作用随**距离衰减**，与起点发生量、终点吸引力**正相关** | 同时考虑供、需、距离衰减 | 模型较复杂 |

**空间阻隔法的三个层次**：① **节点可达性**——相对可达性（两点间距离）与综合可达性（一点到所有其他点的距离总和/平均）；② **网络可达性**——全网所有节点的平均可达程度；③ **基于拓扑**——还考虑连通路径的数量与布局。
（课件七分区案例：a-b 间相对可达性最好，g-f 最差；a 点综合可达性最优。）

**机会累积法的具体形态**：等值线法（按成本分级统计可达目标数）、**移动搜索法**（搜索区内目标数/人口数）、**两步移动搜寻法**（需求点搜索半径内**供需比之和**）及其拓展（距离衰减、可变半径、供需竞争）。课件案例：克服 3 单位阻隔时 a 点触及 76.7% 就业机会、g 点仅 3.3%。

**空间相互作用法：Shen 供需势能模型**（认得结构）：第一步算各区对机会的**需求潜力**（周边居民数 × 距离衰减加和）；第二步算可达性 = Σ[目的地机会数 × 距离衰减 ÷ 该目的地的需求潜力]——**分母引入"竞争"：机会要除以抢它的人**。

## 易错点

1. 泰森多边形两次出场：第 6 讲当**邻近度/服务区划分**工具，第 7 讲当**插值**方法（最近邻点法）——一物二用，论述可点出。
2. Dijkstra **不能负权**；能负权的是 Floyd（两两）和 Bellman-Ford（单源）。
3. 网络服务区靠**网络路径**定边界，缓冲区靠**直线距离**——别混。
4. 定位问题=**布设供应点**（选址），分配问题=**需求分给谁服务**——方向相反。
5. 可达性"没有统一概念"这句话本身是考点——答题先给这句再列三种观点。
6. 机会累积法的致命伤是**阈值依赖**，空间阻隔法的致命伤是**无视吸引力**——各自的"缺点"别张冠李戴。

**本讲在全课程主线中的位置**：前 5 讲的空间关系都建立在"直线世界"（欧氏距离），本讲把度量空间换成**网络**——距离、邻近、服务范围全部重新定义，是空间分析落地城市应用（交通、选址、公共服务）的桥梁。

# 第 7 讲 空间插值

## 【1分钟速览】

**空间插值**：由离散已知点求函数关系，推求区域内任意点的值（有限采样 → 连续曲面）。分类主轴：**确定性插值**（凭距离/平滑假设：趋势面、IDW、样条、泰森、三角剖分、核密度…）vs **克里金插值**（凭**空间自相关结构**=变异函数，做**线性无偏最优估计**）。变异函数三参数：**块金值**（随机性）、**基台值**（变异强度）、**变程**（影响范围）。克里金家族：普通（均值未知）、简单（均值已知）、泛（有趋势）、对数正态（偏态数据）、指示（概率/阈值）、协同（多变量）。经典论述题"IDW vs 样条 vs Kriging 对比"弹药全在本讲。

## 7.1 为什么插值 & 定义分类

**动机**（课件的施肥问题）：想知道全地块土壤养分——密集采样**耗时耗力耗财**且仍是点数据；于是**有选择地采样特征样点 + 空间插值** → 得到**连续曲面**。

**空间插值定义**：对已知空间数据（**离散点**形式或**分区数据**形式），找一个**函数关系式**，使其**最好地逼近已知数据**，并能推求出区域范围内**其它任意点或分区**的值。

**四组分类**（简答常考）：

| 分类轴 | 一边 | 另一边 |
|--------|------|--------|
| 推求范围 | **内插**（同一区域内） | **外推**（其他区域） |
| 数据形式 | **点插值**（点→点） | **面插值/区域插值**（点→面） |
| 用多少样点 | **整体插值**（所有样点，全局特征） | **局部插值**（邻近样点，局部估计） |
| 原理依据 | **确定性插值**（相似性/平滑程度，数学函数） | **克里金/地统计插值**（**空间自相关程度**，线性无偏最优估计） |

再叠一个"**精确 vs 非精确**"：拟合曲面是否必须**通过所有观测点**。

## 7.2 确定性插值：全局非精确类

### 趋势面分析（全局多项式插值）★高频

**本质**：以地理坐标 (x,y) 为自变量，通过**回归分析 + 最小二乘**拟合一个二维（非线性）多项式函数，模拟地理要素的**渐变趋势/宏观分布规律**。全局（用所有样点）+ 非精确（曲面不过所有点）。

**核心原理式**：**实际曲面 = 趋势面 + 剩余面**。趋势面 = 宏观规律（确定性因素）；剩余面 = 局部变化（局部异常、随机干扰、模型误差）。要求：**趋势值最大、残差值最小**。

阶次：一次（平面）、二次、三次多项式趋势面。**阶次困境**（必考）：次数增加 → 曲面越贴近采样点、偏差越小；但方程越复杂 → **越难解释物理意义**，且**阶次越高越容易出现振荡**。

**适度检验三件套**：① **R² 检验**（拟合优度）；② **显著性 F 检验**（模型整体）；③ **逐次检验**（次数增高是否带来**新贡献**——高低阶 SSR 之差做 F 检验）。

（"高阶振荡 → 把区域分块、各块用不同多项式"的启发 = **局部插值法**的由来——课件明说了这条线。）

### 变换函数法与土地利用回归

- **变换函数法（经验方程）**：用**与被预测属性相关的其他属性**建回归方程做整体插值——除坐标外引入高程、距离、地形因子等。例：冲积平原土壤重金属含量 ~ 距河流距离 + 高程（因子**便于获取**且**有物理意义**）。
- **土地利用回归（LUR）**：变换函数法的典型代表——模拟**城市大气污染物浓度**空间分异的通用模型。因变量=监测点污染物浓度；自变量=站点周边**土地利用/道路交通/人口密度/气象**等 → 建回归 → 全区均匀布点算模拟值 → 插值成面。对比方案的短板（附加题料）：MODIS 遥感反演时空分辨率低；大气扩散模拟复杂难推广。

## 7.3 确定性插值：局部非精确类

| 方法 | 思想 | 关键点 |
|------|------|--------|
| **移动拟合法**（单点移面内插） | 以**待插点为中心**定义局部数学模型拟合周围点，范围随待插点**移动** | 邻域要覆盖极值；一般取 **4~12 个采样点**，不规则分布多取些 |
| **局部多项式法**（移动内插法） | 移动拟合基础上**考虑距离权重**，移动窗口拟合多个多项式 | **小窗口**强化近数据、**大窗口**强化远数据；适于高程起伏大、局部变化明显 |
| **核密度估计 KDE** | 用平滑峰值函数（核）拟合样本，**非参数**估计概率密度 | **核函数选取的影响远小于带宽 h**；h 越大越平滑（过大抹平、过小毛刺）；带宽平衡**方差与偏差**（MISE 最小化）；无需先验分布假设；有边界效应；应用：犯罪热点、建筑密度 |

（又见带宽的偏差-方差权衡——和 GWR 第 5 讲同款逻辑，跨章对比素材。）

## 7.4 确定性插值：局部精确类 ★对比题主战场

| 方法 | 思想 | 优点 | 缺点/适用 |
|------|------|------|----------|
| **泰森多边形法**（最近邻点法/边界内插法) | 假定重要变化都发生在**边界上**，边界内**均匀同质**；待求点取所在多边形参考点的值 | 无前提条件、简单高效 | **插值精度低**、受样本点影响大；适于**小区域、变异不明显** |
| **三角剖分法** | 参考点连成 **Delaunay 三角网**（最小角最大化、外接圆不含他点、唯一），三角形内**线性插值**（三顶点加权） | 简单 | 每个预测值**只由 3 个实测值**得到、**无误差估计**；三角形边上**斜率突变** |
| **反距离加权 IDW** | 以插值点与样本点**距离的倒数幂**为权重的加权平均；幂 p 越大远点权重越小（p 常取 2，可由 RMSE 定） | 适合变化大的数据、结果易解释 | **极值只出现在采样点上**；权重选择敏感；易产生**孤立点/多极中心**（牛眼）；要求样点分布均匀且足够密集 |
| **样条函数法** | 分段函数逐段拟合、相邻分块边界 n−1 次导数连续（曲面光滑）；薄板样条=表面二阶导平方和最小（**最小曲率**） | **精确**（过每个点）、保留细微特征、插值快、观测点处**可导** | 适于**平缓渐变**表面（高程、气象、水污染）；区域化变量波动大时不宜；分规则样条（可超出样本值域，更平滑）与张力样条（受值域约束更严格） |

**IDW 的理论出身**：它就是 **Tobler 第一定律的直接机械化**——近者权重大。这句话是论述题点题金句。

## 7.5 克里金插值 ★论述必考

> 📜 **历史锚点**：南非**地质学家 Krige（1951）**在金矿品位估计中提出雏形，法国数学家 **Matheron（1976 前后）**将其理论化、命名为 Kriging，创立**地统计学**。它为什么赢过 IDW 这类"拍距离脑袋"的方法：**权重不再由距离公式主观指定，而是由数据自己的空间自相关结构（变异函数）估计出来**，且能给出**预测误差（方差）**——既最优又自知误差。IDW 输在权重形式是人定的、无误差估计。

**定义**：利用**区域化变量**的原始数据和**变异函数**的结构特点，对未采样点的区域化变量取值进行**线性无偏最优估计**（BLUE）的方法。
与 IDW 对比（课件原话）：相同点——都是**局部估计的加权平均**；不同点——**权重的确定**：IDW 基于距离，Kriging 基于**变异函数（半方差分析）**。

### 理论地基一：区域化变量

**区域化变量**：呈空间分布的变量。**双重性格：既有随机性又有结构性**（必背）：
- **随机性**：局部的、随机的、异常的
- **结构性**：点 x 与 x+h 处的值**具空间自相关性**
- 另有：**空间局限性**（一定范围外属性为 0）、不同程度的**连续性**、**各向异性**（各方向性质变化不同）

**区域化变量 vs 普通随机变量**（对比表可考）：取值随**位置**而定 vs 只随概率分布；**不可重复观测**（同一位置不能取两次样）vs 可无限重复；样本间**空间自相关** vs 要求相互独立。

### 理论地基二：三大平稳假设（认得层级）

| 假设 | 内容 | 强度 |
|------|------|------|
| 平稳假设 | 随机函数性质不随位置移动而改变（平移不变） | 最严 |
| **二阶平稳** | ① 均值为常数；② 协方差只依赖距离和方向 | 中 |
| **本征假设**（内蕴假设） | ① 增量的期望为 0；② **增量的方差**只依赖距离和方向 | 最宽松，**普通克里金的前提** |

### 理论地基三：变异函数 ★名词解释最高危

**协方差函数** C(h)：区域化变量相距 h 两点的协方差；h=0 时=方差。
**变异函数（半变异函数）γ(h)**：任一方向上，相距 h 的两个区域化变量值 Z(x) 与 Z(x+h) 的**增量的方差**（的一半）。性质：偶函数、非负、h→∞ 时趋于先验方差。

**结构分析三参数**（背熟+会画那条上升后走平的曲线）：

| 参数 | 定义 | 功能读法 |
|------|------|---------|
| **变程 Range (a)** | h 超过 a 后 γ(h) 不再增大（"跃迁现象"）的临界距离 | **影响范围**——超出变程，空间自相关消失 |
| **基台值 Sill (C₀+C)** | γ(h) 稳定后的水平值 | **变异强度** |
| **块金值 Nugget (C₀)** | h→0 时 γ(h) 的非零截距（块金效应） | **随机性**——来源：小于采样尺度的微观变异 + 采样/测量/分析误差 |

（原点处的性状反映**空间连续性**；不同方向的变异函数图不同 = **各向异性**——"各向同性是相对的，各向异性是绝对的"。）

**理论模型**认名字 + 一句选型：有基台——**球状**（影响范围小、采样点多，最常用）、**指数**（影响范围大、采样点少）、**高斯**（原点抛物线、高度连续，如矿层厚度）、有基台线性、纯块金效应；无基台——幂函数、线性、对数；**孔穴效应模型**（γ 周期波动）。

### 克里金家族 ★背这张总表

| 方法 | 前提/特征 | 一句话 |
|------|----------|--------|
| **普通克里金 OK** | **本征假设；均值未知常数** | 主力型号；权重和=1 保证无偏；**精确插值**（估到实测点=原值） |
| **简单克里金 SK** | **二阶平稳；均值已知常数** | 均值信息使预测方差稍小；但均值实际难知，**很少直接用**，常内嵌于指示/协同克里金 |
| **泛克里金 UK** | 数据**存在主导趋势**（非平稳）：Z(x)=m(x)+ε(x) | 先拟合趋势 → 残差做普通克里金 → 相加；结果一般优于 OK；是否平稳取决于**观测尺度** |
| **对数正态克里金** | 数据**严重偏斜非正态** | 取对数 → 近正态 → OK/SK → 回转公式还原 |
| **指示克里金 IK** | 关心**是否超过阈值/概率**（如土壤重金属超标） | 数据转 0/1 指示码 → 结果为 0~1 的**条件概率**；**非线性、非参数**：无需正态假设、不怕异常值、无需变换 |
| **协同克里金 CK** | **多个相关的区域化变量**（协同区域化），建**交叉变异函数** | 用**易观测变量弥补难观测变量**：气温与海拔负相关，缺气温可用海拔补 |

**普通克里金的估计逻辑**（论述题能讲出三层就赢）：预测值 = 邻域实测值的**线性**加权（局部线性）→ 权重满足和为 1（**无偏**：期望误差为 0）→ 在无偏约束下**最小化预测方差**（**最优**，拉格朗日乘子法解方程组）。三个词连起来就是"线性、无偏、最优（+方差已知）"。

## 易错点

1. **变异函数三参数**必须一字不错：变程=影响范围、基台=变异强度、块金=随机性（"块金"是 h→0 的截距，不是曲线顶）。
2. **IDW 极值只能出现在采样点**（做不出比观测更高的峰）；**样条可以超出样本值域**——两者正相反。
3. Kriging vs IDW 的分野在**权重来源**（变异函数 vs 距离函数）和**有无误差估计**，不是"一个准一个不准"。
4. 普通克里金前提是**本征假设**，简单克里金是**二阶平稳+均值已知**——别互换。
5. 趋势面分析是**全局非精确**；泰森/三角剖分/IDW/样条是**局部精确**——分类轴要摆对。
6. "确定性插值 vs 地统计插值"的分类依据：前者用**相似性/平滑程度**，后者用**空间自相关程度**——这句是定义级考点。
7. 指示克里金输出的是**概率**（0~1），不是属性值本身。

**本讲在全课程主线中的位置**：全课程的"集大成"——Tobler 定律（第1讲）给它合法性，回归（第5讲）给了趋势面和 LUR，泰森多边形（第6讲）在这儿转正为插值法，而克里金用"变异函数"把**空间自相关（第4讲）从诊断工具升级为生产工具**：别人用自相关发现问题，克里金用自相关解决问题。
