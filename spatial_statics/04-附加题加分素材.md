# 附加题加分素材

> 定位：考试"专业实在"——素材侧重**应用案例**与**方法评价**，历史只做一句话锚点。
> 用法：论述题结尾升华、附加题填肉。每条都能独立成段。

---

## 一、方法评价速查（每方法一句优点一句缺点）

| 方法 | 一句优点 | 一句缺点 |
|------|---------|---------|
| K-means | 计算复杂度低，大样本可行 | 需预先给 k，对离群点非常敏感，只适合成团分布 |
| 层次聚类 | 免指定类数，输出多粒度层次结构 | 计算与空间复杂度很高，难上大数据 |
| 决策树 | 决策过程可显式解释，无分布假设 | 对训练样本敏感、极易过拟合，须剪枝 |
| Kappa 系数 | 扣除偶然一致，抗类别不平衡 | 仍是单一数值，掩盖各类别的具体错分方向 |
| Pearson | 直观、检验体系成熟 | 要正态、只测线性、怕极端值 |
| Spearman/Kendall | 免正态、测单调、抗离群 | 损失原始量值信息 |
| Moran's I | 一个数概括全区聚集程度 | 分不清高值聚集还是低值聚集，且掩盖局部差异 |
| Gi\* 热点分析 | 直接输出分置信度热冷点图，决策友好 | 依赖距离阈值/权重矩阵的设定 |
| OLS 回归 | 简单、可解释、检验完备 | 空间数据违反独立性假设，残差空间聚集时失效 |
| SLM/SEM | 显式建模空间依赖，提升拟合并还原机制 | 全局系数；不能 OLS 估计，须 ML/GMM；依赖 W 的设定 |
| GWR | 一地一方程，最大限度展示空间异质性 | 带宽选择敏感（偏差-方差权衡）；参数多、易过拟合 |
| 缓冲区 | 简单直观 | 直线距离忽视真实路网阻隔 |
| Dijkstra | 经典可靠、可有向图 | 不能负权，大网络效率低 |
| 空间阻隔可达性 | 计算简单直观 | 忽略目的地吸引力与人的活动 |
| 机会累积可达性 | 符合直觉、同范围内可比 | 阈值依赖，无法反映区域体系地位 |
| IDW | 快、易解释 | 权重人为定、无误差估计、牛眼效应 |
| 样条 | 光滑、精确、保细节 | 波动大的变量不适用 |
| 趋势面 | 提炼宏观趋势、检测最大偏离 | 高阶振荡、物理意义难解释 |
| 克里金 | 线性无偏最优+自带误差估计 | 需平稳假设与变异函数建模，流程重 |

## 二、应用案例库（偏城市规划，优先用课件自带案例）

### 课件原生案例（首选引用，显得"上过课"）

1. **John Snow 伦敦霍乱（1854）**：病死者定位+水井叠加 → 锁定宽街水泵。用途：空间分析价值的万能开场/收尾。
2. **深圳各区常住人口描述统计**：标准差 37.7 万人、CV=84.4% → 深圳人口空间高度不均。用途：描述统计应用、引出空间不均衡议题。
3. **深圳 TAZ 工作日人口密度空间自相关**：全局 Moran's I 显著为正（p≈0）→ 局部 Moran + Gi\* 分置信度热冷点图。用途：全局→局部标准作业流的实例。
4. **美国南部县室内极端案件（犯罪）空间回归**：OLS R²=0.104 → LM 检验（Robust LM-lag 显著）→ SLM R²=0.159、残差自相关≈0。用途：空间回归选型全流程实例。
5. **美国膀胱癌死亡率 GWR**：OLS R²=0.25 → GWR R²=0.52，系数图揭示局部负相关。用途：空间异质性"被平均掩盖"的铁证。
6. **佐治亚州学士学位比例 GWR**：0.24→0.30，AIC 选带宽 49.46。用途：GWR 第二例证。
7. **深圳义务教育段中小学选址**：公共服务均等化评估预警（网络分析定位问题）。用途：定位配置分析+本校研究方向，答"空间分析在城市规划中的应用"首选。
8. **消防站辖区分配**：道路距离为权、同时广度搜索、容量约束。用途：分配问题实例。
9. **土地适宜性评价**：多栅格重分类+叠置运算。用途：叠加分析实例。
10. **LUR 模拟城市大气污染**：污染浓度 ~ 土地利用/交通/人口 → 全域插值。用途：插值+回归的交叉应用。
11. **胡焕庸线**：中国人口地理分界。用途：空间异质性/空间分布不均的中国经典例。

### 可补充的通用案例（一句话点到即可）

- 房价空间分析：房价的空间自相关（SLM——邻里房价传导）与影响因素的空间异质（GWR——学区变量在不同城区系数不同）。
- 疫情传播：意大利→法英 vs 韩国（课件自带的空间依赖例子），SIR+空间权重。
- 城市热岛：温度插值（克里金）+ 与绿地覆盖的 GWR。
- 共享单车/网约车热点：KDE + Gi\*，指导投放调度。

## 三、"升华句"仓库（论述题结尾用，每题选一句）

1. 空间统计的核心贡献，是把 Tobler 第一定律从一句格言变成了可计算、可检验、可利用的模型构件。
2. 空间相关性对经典统计是"违反假设的麻烦"，对空间统计是"可以借力的资源"——克里金和空间滞后模型的预测能力正来源于此。
3. 全局指标回答"有没有"，局部指标回答"在哪里"，回归模型回答"为什么"，插值回答"没测的地方是多少"——四类方法合起来才是完整的空间分析链。
4. 方法没有优劣，只有假设是否匹配数据：正态与否选相关系数，依赖或异质选 SAR 或 GWR，平稳与否选普通或泛克里金。
5. 从 John Snow 的手绘地图到 GeoDa/PySAL 的一键计算，变的是算力，不变的是"把数据放回空间中去理解"的思想。
6. 空间权重矩阵是整门课的隐形主角——定义了"谁是邻居"，就定义了模型能看见怎样的空间。

## 四、一句话历史锚点（点到为止，不展开）

- 正态分布：高斯 1809 从误差理论导出；20 世纪初小样本革命催生 t/F/卡方。
- "回归"：高尔顿遗传研究，身高"向均值回归"。
- Tobler 第一定律：1970。
- Dijkstra：1959。
- 泰森多边形：气象学家 Thiessen，为估算降水分布而生。
- 克里金：南非矿业工程师 Krige（1951）金矿品位估计 → 法国 Matheron 理论化，地统计学由此创立。
- 空间大数据：2011 年 Science 专刊宣告大数据时代；80% 数据与空间位置相关。
