# 摄影测量学期末复习资料专区

本目录用于摄影测量学期末复习。当前采用“双层资料”结构：

- `00_总复习提纲与自测题.md`：总纲、公式速查、易混点、自测题，考前优先看。
- `detailed/`：用户提供的 Claude 详细版讲义，作为正式学习底稿，特点是细、全、接近考试复习提纲。
- `simple/`：Hermes 早期整理的简版讲义，适合第一次建立概念框架。
- `exam-focus.md`：根据 2024 题目重点整理的题型清单和背诵优先级。

## 推荐学习顺序

1. 先看 `00_总复习提纲与自测题.md`，知道全课程主线。
2. 每一讲学习时，以 `detailed/` 为主。
3. 如果某一讲太难，再回到 `simple/` 看简版解释。
4. 学完一讲后做该讲自测题，由 Hermes 老师模式逐题追问、纠错和压缩成考试答案。
5. 考前最后看 `exam-focus.md` 和总复习提纲中的表格。

## 2024 题目重点提醒

题型：概念、简答、论述、附加题。

重点包括：摄影测量学、特征匹配、光束法空三、航向倾角、采样与量化、中心投影、相对/绝对航高、影像特征、后方交会控制点数量、Moravec、前方交会三种方法、三大基本原理、自由航带网、Harris、SIFT。

> 注意：如果讲义、PPT、老师课堂表述不一致，考试时优先采用老师 PPT/课堂表述。Hermes 负责讲解、核对和压缩成答题模板。
