# 第九讲 密集匹配

## 1. 本讲定位
特征匹配通常只得到稀疏同名点；密集匹配则希望为影像中大量像素寻找对应点，从而生成视差图、DSM/DEM或三维点云。

## 2. 密集匹配定义
密集匹配是在具有重叠的多幅影像中，为尽可能多的像素或小区域寻找同名点，获取连续视差信息或三维信息的过程。

## 3. 与特征匹配的区别
| 项目 | 特征匹配 | 密集匹配 |
|---|---|---|
| 匹配对象 | 稀疏特征点 | 大量像素/区域 |
| 输出 | 稀疏同名点 | 视差图/稠密点云 |
| 用途 | 定向、配准、空三 | DSM/DEM、三维重建 |
| 难点 | 特征稳定与误匹配 | 遮挡、弱纹理、重复纹理、计算量 |

## 4. 基本约束
密集匹配常利用：
1. **灰度相似性约束**：同名区域灰度或纹理相似；
2. **核线约束**：对应点位于核线上，降低搜索维度；
3. **连续性约束**：相邻像素视差通常连续；
4. **唯一性约束**：一个像点通常对应一个同名点；
5. **顺序约束**：沿核线匹配顺序一般保持一致。

## 5. 基本流程
1. 影像预处理与几何校正；
2. 确定核线几何或搜索范围；
3. 选择匹配窗口或代价函数；
4. 逐像素/逐区域计算匹配代价；
5. 代价聚合与视差优化；
6. 误匹配剔除与空洞填补；
7. 生成视差图、深度图或三维点云。

## 6. 主要困难
1. 弱纹理区域缺乏明显匹配信息；
2. 重复纹理容易产生歧义；
3. 遮挡区域在另一张影像中不可见；
4. 光照差异导致灰度不一致；
5. 地物边缘处视差不连续；
6. 大范围密集计算量较大。

## 7. 与摄影测量产品的关系
密集匹配可生成密集三维点云，再进一步生产：
- DSM/DEM；
- DOM正射纠正所需高程数据；
- 三维模型；
- 建筑物或地形表面模型。

## 8. 考试答题模板

### 名词解释：密集匹配
密集匹配是指在多幅重叠影像中，为大量像素或小区域寻找同名点，获得连续视差或深度信息的方法。它可用于生成稠密点云、DSM/DEM和三维模型。

### 简答：密集匹配与特征匹配区别
特征匹配主要在角点、兴趣点等少量稳定特征之间建立对应关系，输出稀疏同名点，常用于影像定向和配准；密集匹配则为大量像素或区域寻找对应点，输出稠密视差图或点云，常用于三维重建和DEM/DSM生成。密集匹配信息更丰富，但更容易受到弱纹理、遮挡、重复纹理和计算量影响。
