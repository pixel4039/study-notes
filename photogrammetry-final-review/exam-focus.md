# 摄影测量学考试重点整理

## 题型
老师说明题型为：
1. 概念；
2. 简答；
3. 论述；
4. 附加题。

## 2024题目重点

### 一、名词解释
1. 摄影测量学
2. 特征匹配
3. 光束法空中三角测量
4. 航向倾角
5. 采样与量化

### 二、简答题
1. 中心投影的特征。
2. 作图说明相对航高与绝对航高，并说明关系。
3. 影像特征的概念是什么？举例说明影像特征类型。
4. 单像空间后方交会一般需要几个控制点？说明理由。
5. 作图说明 Moravec 算子的原理。

### 三、论述题
1. 立体空间前方交会有三种主要方法，阐述每一种方法的主要思想，并列出主要步骤。
2. 摄影测量学三大基本原理：列出每个基本原理的公式，并说明几何含义和作用。

### 四、附加题
1. 说明自由航带网的构成。
2. Harris 算子的基本原理及特征。
3. SIFT 算子相比 Harris 多了什么特征？说明理由。

## 建议背诵优先级

### 第一优先级：24题目重点
这些题最应该先达到“能闭卷写出来”的程度。

### 第二优先级：期末复习PDF中的标准概念
尤其是第二章基础概念、坐标系、方位元素、前后方交会、空三。

### 第三优先级：PPT中的公式和图示
特别是共线方程、航高示意、中心投影示意、Moravec/Harris原理图。

## 答题策略

### 概念题
结构：定义 + 关键词 + 作用。

### 简答题
结构：先总述，再分点，每点一句解释。

### 论述题
结构：
1. 先说明问题背景；
2. 给出基本思想；
3. 分步骤；
4. 最后说明作用或优缺点。

### 附加题
附加题通常考区分度。不要只背定义，要说明“为什么”：例如 SIFT 为什么比 Harris 更适合尺度变化影像，因为它引入尺度空间、主方向和描述子。
