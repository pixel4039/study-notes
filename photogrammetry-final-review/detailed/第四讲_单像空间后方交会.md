# 第四讲 单像空间后方交会

> 四大板块：**① 影像内定向 ② 单像空间后方交会 ③ 共线方程线性化 ④ 解求外方位元素**。这是共线方程"应用②（求外方位元素）"的具体实现，**必考大题**：概念、误差方程、法方程、解算步骤、精度评定。

---

## 一、影像内定向

### 问题的提出
- 要从影像提取物体空间信息，须先确定像点坐标。
- 解析摄影测量中量测的是**像片框标坐标**，放置像片未严格对准，**框标坐标系与以像主点为原点的像平面坐标系不重合**。
- 数字摄影测量中量测的是**扫描坐标系**坐标，同样与像平面坐标系不重合。

### 基本概念与目的
- **影像内定向**：利用**平面相似变换**，将像片框标坐标（或扫描坐标）转化到**以像主点为原点的像平面直角坐标系**中。
- **目的**：**恢复影像的内方位元素**，确定量测坐标系与像平面坐标系的关系，以及影像可能存在的变形。
- 作业依据：影像的**框标**。相机检校提供框标的理论坐标，量测得到框标在量测坐标系的坐标，解算两坐标系关系。

### 内定向变换公式（多项式变换，(x,y)理论坐标，(I,J)量测坐标）
- **线性正形变换**（4参数：平移2+旋转1+尺度1）：$x = a_0 + a_1 I - a_2 J,\ y = a_0' + a_2 I + a_1 J$
- **仿射变换**（6参数，最常用）：$x = a_0 + a_1 I + a_2 J,\ y = b_0 + b_1 I + b_2 J$
- **双线性变换**（含 IJ 项）、**投影变换**（含 I², J² 项）。

### 补充：普通数码相机畸变
- 相机模型：内参矩阵 + 外参矩阵。
- **径向畸变**：
  - **桶形畸变**：光轴中心放大率 > 边缘，常见于**广角/鱼眼镜头**。
  - **枕形畸变（鞍形）**：边缘放大率 > 中心，常见于**远摄镜头**。
- **切向畸变**：透镜与成像面不平行引起。畸变校正目的是恢复正确成像几何。

## 二、单像空间后方交会（★核心概念）

### 问题的提出
- 已知每张像片的 6 个外方位元素，就能恢复像片与地面的关系，重建立体模型。获取外方位元素两种方法：
  - **方法一**：用 GNSS + IMU（POS）直接测定外方位元素。
  - **方法二（本讲）**：利用一定数量的**地面控制点**，根据**共线方程反求**像片外方位元素——即**单张像片的空间后方交会**。

### 定义
- 利用**至少 3 个已知地面控制点** A(X_A,Y_A,Z_A)、B、C 及其对应像点 a(x_a,y_a)、b、c，确定**一张像片的 6 个外方位元素**的方法。以单张像片为基础，故称单像空间后方交会。

### 控制点类型
- 平面控制点：已知 (X, Y)；高程控制点：已知 Z；**平高控制点**：X,Y,Z 都已知。
- **单像空间后方交会使用平高控制点**。

### 为什么至少需要 3 个控制点？
- 未知数 6 个（外方位元素）。每个控制点提供 2 个方程（x, y），3 个点 → 6 个方程，恰好解 6 个未知数。
- 实际为提高精度和检核，一般用 **4 个以上**控制点（多余观测 → 最小二乘平差）。
- 属于**间接平差**（观测值 = 像点坐标，未知数 = 外方位元素改正数）。

## 三、共线方程线性化（★推导，可能考误差方程）

### 已知/观测/未知
- **已知值**：内方位元素 x₀,y₀,f，比例尺分母 m，物点坐标 (X_A,Y_A,Z_A)。
- **观测值**：像点坐标 x, y（改正数 v_x, v_y）。
- **未知数**：外方位元素 X_S,Y_S,Z_S,φ,ω,κ。

### 线性化（泰勒级数展开取一次项）
共线方程是外方位元素的**非线性函数**，须用泰勒级数在近似值处展开，只取一次项：
$$v_x = a_{11}dX_S + a_{12}dY_S + a_{13}dZ_S + a_{14}d\varphi + a_{15}d\omega + a_{16}d\kappa - l_x$$
$$v_y = a_{21}dX_S + a_{22}dY_S + a_{23}dZ_S + a_{24}d\varphi + a_{25}d\omega + a_{26}d\kappa - l_y$$
- 常数项：$l_x = x - (x),\ l_y = y - (y)$，其中 (x)、(y) 是把外方位元素近似值代入共线方程算得的像点坐标近似值。
- $a_{ij}$ 为共线方程对各外方位元素的**偏导数**（误差方程系数）。
- 线元素偏导数含 $\frac{1}{\bar{Z}}$ 等（$\bar{Z}=a_3(X_A-X_S)+b_3(\cdots)+c_3(\cdots)$），角元素偏导数较复杂。

### 误差方程与法方程（矩阵形式）
- 误差方程：$V = AX - L$（A 系数阵，X 未知数改正数向量，L 常数项）。
- 最小二乘准则：$V^T P V = \min$（P 为权阵，像点等精度量测时 P = 单位阵）。
- 令一阶导为零 → **法方程**：
$$A^T P A X = A^T P L \quad\Rightarrow\quad X = (A^T P A)^{-1} A^T P L$$

## 四、解求外方位元素（★完整步骤，高频简答）

### 空间后方交会解算流程
```
读入已知数据 → 确定外方位元素初值 → 组成误差方程 → 最小二乘解算
→ 计算外方位元素改正数 → 改正数<限差? →(否,迭代)→ ...(是)→ 输出结果
```

### 完整计算步骤（8步）
1. **获取已知数据**：比例尺 1/m；内方位元素 x₀,y₀,f；控制点地面测量坐标 (Xt,Yt,Zt) 转为地面摄影测量坐标 (X,Y,Z)。
2. **量测控制点像点坐标**：控制点标刺在像片上，量测框标坐标，经像点坐标改正得 x, y。
3. **确定未知数初值**（竖直摄影假设）：
   - 角元素：$\varphi_0 = \omega_0 = \kappa_0 = 0$。
   - 线元素：$Z_{S0} = H = m\cdot f$；$X_{S0} = \frac{1}{n}\sum X_{ti}$，$Y_{S0} = \frac{1}{n}\sum Y_{ti}$（控制点坐标均值）。
4. **计算旋转矩阵 R**（用角元素近似值）。
5. **逐点计算像点坐标近似值** (x)、(y)，得常数项 $l_x=x-(x),\ l_y=y-(y)$。
6. **组成误差方程并法化**（逐点计算系数和常数项）。
7. **解求外方位元素改正数**，加到近似值上得新值：$X_S = X_{S0}+dX_S+\cdots$（迭代累加）。
8. **检查是否收敛**：改正数与限差比较，小于限差则完成；否则以新值为近似值重复 ④–⑧，直到收敛。

### 精度评定（★易考公式）
- **单位权中误差**：$m_0 = \sqrt{\dfrac{V^T P V}{2n - 6}}$
  - 为什么是 **2n − 6**？n 个控制点提供 **2n 个方程**（每点 x、y 两个），减去 **6 个未知数**（外方位元素）= 多余观测数（自由度）。
- **未知数协因数阵**：$Q_{xx} = (A^T P A)^{-1}$。
- **未知数中误差**：$m_{x_i} = m_0\sqrt{Q_{ii}}$（$Q_{ii}$ 为协因数阵主对角元素）。

## ★本讲高频考点速记
1. 内定向：用**平面相似变换/仿射变换**将框标(扫描)坐标→像平面坐标，目的是恢复内方位元素。
2. 后方交会定义：≥3 个**平高控制点** + 对应像点 → 求 1 张像片的 6 个外方位元素。
3. 属**间接平差**；线性化用**泰勒级数取一次项**。
4. 误差方程 $V=AX-L$，法方程 $A^TPAX=A^TPL$，解 $X=(A^TPA)^{-1}A^TPL$。
5. 初值：角元素=0，$Z_{S0}=mf$，$X_{S0}/Y_{S0}$=控制点坐标均值。
6. 8步解算流程 + 迭代收敛条件（改正数<限差）。
7. 精度：单位权中误差分母 **2n−6**（能解释原因）。
