# 第七讲 影像特征提取与匹配（SIFT算子）

> 本讲为"part2"，核心是 **SIFT 算子**（提取4步 + 匹配2步）及**影像匹配基础概念**。SIFT 的四大步骤、DoG、128维描述子、RANSAC 是**必考点**。（part1 的 Moravec/Harris/Förstner 兴趣算子未在本文件夹课件中，若考请另找 part1。）

---

## 一、特征提取面临的挑战与不变性
- 特征提取需应对：**几何变化**（平移、缩放、旋转、仿射）+ **灰度变化**（光线、曝光量、感光度）。
- 选取特征算子需考虑**四种基本不变性**：
  - **旋转不变性、尺度不变性、亮度（光照）不变性、仿射不变性**。

---

## 二、SIFT 算子概述（★）
- **SIFT = Scale Invariant Feature Transform（尺度不变特征变换）**，David G. Lowe 1999年提出、2004年完善。
- **要解决的问题**（针对 Harris 缺陷）：Harris **不具尺度不变性**、无法准确描述兴趣点 → SIFT 开发**具尺度和旋转不变性的检测算子** + **鲁棒的特征描述符**。
- **实质**：在不同尺度空间查找关键点并计算方向，将影像映射为**局部特征向量集**；对**平移、缩放、旋转**不变，对**光照、仿射、投影**变化有一定不变性。
- 可解决：目标 RST（旋转/缩放/平移）、视点仿射/投影变换、光照、遮挡、杂物场景、噪声。

### SIFT 工作过程（提取4步 + 匹配2步 = 6步）
① 尺度空间兴趣点检测 → ② 兴趣点定位 → ③ 主辅方向分配 → ④ 兴趣点描述 ｜ ⑤ 同名兴趣点匹配 → ⑥ 消除错误匹配。

---

## 三、SIFT 特征提取的 4 个步骤

### ① 尺度空间兴趣点检测（★核心）
- **兴趣点**：十分突出、不因光照改变而消失的点（角点、边缘点、暗区亮点、亮区暗点）；是不同尺度空间下检测出的**具方向信息的局部极值点**；有三个特征：**尺度、方向、大小**。
- **尺度空间**：指影像的**模糊程度**（非大小）。从近到远看物体越来越模糊 = 尺度越来越大。尺度空间满足视觉不变性（灰度/对比度/平移/尺度/仿射不变）。
- **建立方法**：影像金字塔（不同分辨率影像序列）。金字塔化 = 平滑 + 降采样。
- **★高斯核**：高斯核是**唯一能产生多尺度空间的核**。尺度空间 $L(x,y,\sigma) = G(x,y,\sigma) * I(x,y)$（原始影像与可变尺度2维高斯函数卷积），σ 为尺度。
  - 高斯核性质：低通滤波（去高频）；两高斯卷积仍为高斯；**多次小方差卷积 = 一次大方差卷积**；用 σ 卷积两次 = 用 $\sqrt2\sigma$ 卷积一次（$\sigma_新=\sqrt2\sigma_旧$）；可分离（分解为两个一维高斯乘积）；中心/圆对称；权重和=1。
  - 经验法则：卷积核半窗宽 = 3σ，模板尺寸 = **2×3σ+1**。
- **高斯金字塔**：由**层组（Octave）**和**层（Interval）**构成。先降采样得不同分辨率，再对每层高斯卷积。
  - $\sigma(s) = \sigma_0 \cdot 2^{s/S}$，组数 $O = \log_2[\min(M,N)] - 3$，$k = 2^{1/S}$，S 一般 3~5。
  - 后一组第一层影像由前一组倒数第三层影像降采样（1/2）得到。
- **★LoG 与 DoG**：
  - **LoG（高斯拉普拉斯，Laplacian of Gaussian）**：尺度规范化的 LoG 算子具有真正的尺度不变性。
  - **DoG（高斯差分，Difference of Gaussian）**：$D(x,y,\sigma) = [G(x,y,k\sigma)-G(x,y,\sigma)]*I = L(x,y,k\sigma)-L(x,y,\sigma)$。
  - **DoG ≈ LoG**（$LoG \approx \frac{DoG}{\sigma(k-1)}$），DoG 只需**相邻尺度高斯影像相减**，计算简化！
  - 高斯金字塔每组 **S+3 层**影像，DoG 金字塔每组 **S+2 层**。
- **★DoG 局部极值点检测**：每个像素与**同尺度 8 个相邻点 + 上下相邻尺度各 9 个点 = 26 个点**比较，若为极大或极小则为候选兴趣点（保证在尺度空间和二维空间都是极值）。

### ② 兴趣点定位
- 离散极值点 ≠ 真极值点 → 用**三维二次函数（Taylor 展开）拟合**精确定位（**子像素插值 Sub-pixel Interpolation**），Lowe 程序迭代 5 次。
- 去除低对比度点、去除**边缘响应点**（DoG 有强边缘效应）：用 **2×2 Hessian 矩阵**计算主曲率，边缘点在横跨方向主曲率大、垂直方向小，据此剔除。

### ③ 兴趣点方向分配（★旋转不变性来源）
- 目的：尺度不变性→缩放不变；**利用邻域梯度方向分布→旋转不变**。
- 梯度幅值 $m(x,y) = \sqrt{(L(x+1,y)-L(x-1,y))^2 + (L(x,y+1)-L(x,y-1))^2}$，梯度方向 $\theta = \arctan\frac{L(x,y+1)-L(x,y-1)}{L(x+1,y)-L(x-1,y)}$。
- **梯度方向直方图**：36 柱（0~360°，每10°一柱），由半径 3×1.5σ 区域生成，两次平滑。
- **主方向**：直方图主峰值（=特征点方向）。**辅方向**：达到主峰值 **80%** 能量的其它峰值（约15%兴趣点有多方向，增强匹配鲁棒性）。
- 将窗口旋转到主方向 → 特征矢量具有**旋转不变性**。

### ④ 兴趣点描述（★128维描述子）
- 检测+方向分配后每个兴趣点有三个信息：**位置、尺度、方向**（→ 平移/缩放/旋转不变）。
- **描述子（Descriptor）**：对兴趣点周围区域分块、算块内梯度直方图，生成唯一性向量。
- **Lowe 的 SIFT 描述子 = 4×4×8 = 128 维**（综合不变性与独特性最优）：
  - 取 **16×16** 邻域 → 划分 **4×4 个 4×4 子窗口** → 每子窗口算 **8 方向**梯度直方图 → 每个子窗口=1个种子点 → 16 个种子点 × 8 方向 = **128 维**。
- **描述子规范化**：先对每方向幅值设阈值（一般 0.2）截断，再归一化为单位向量 → 去除**光照影响、灰度整体漂移**。

## 四、SIFT 特征匹配的 2 个步骤

### ⑤ 兴趣点匹配
- 对参考图像与目标图像各建描述子集合，用**欧氏距离**度量 128 维描述子相似性：$d(R_i,S_j) = \sqrt{\sum_{k=1}^{128}(r_{ik}-s_{jk})^2}$。
- **匹配策略**：**最近邻距离比 < 阈值**（最近距离/次近距离 < Threshold，一般 0.6~0.8）才认为匹配。
- 搜索方法：穷举匹配（暴力/蛮力匹配）、**K-D tree**（加速）。

### ⑥ 消除错误匹配（★RANSAC）
- 遮挡、变形、重复/弱纹理会产生大量错配点。
- **RANSAC（Random Sample Consensus，随机抽样一致）**：鲁棒参数估计方法，反复测试迭代。
  - 基本思想：设计目标函数 → 反复提取最小点集估计参数初值 → 将数据分为**内点（inlier）/外点（outlier）** → 用所有内点重新估计参数。
- **RANSAC 计算单应矩阵 H 消除误匹配步骤**：
  1. 随机选 **4 对**匹配特征；
  2. 用 **DLT（直接线性变换）**计算单应矩阵 H（唯一解）；
  3. 对所有匹配点计算映射误差 ε；
  4. 按误差阈值确定 inliers，迭代直到找到**最大 inliers 集合**；
  5. 用最大 inliers 集合重新计算 H。

## 五、影像匹配基础概念（★与第八讲衔接）
- **配准/匹配/几何校正**区别：
  - **配准（registration）**：两幅影像间的对齐。
  - **匹配（matching）**：**寻找同名特征（点）的过程**。
  - **几何校正**：按几何变换关系对辅影像逐像素处理变为配准影像。
- **影像匹配定义**：以影像相关手段用计算机代替人眼进行立体观察，自动确定同名像点的过程。是**自动生成 DSM、三维重建的基础**。
- **共轭实体**：目标空间特征的影像（点、线、面等）。**匹配实体**：用于比较确定共轭实体的要素（灰度值分布、特征向量等）。

| 共轭实体 | 匹配实体 | 相似性测度 |
|---|---|---|
| 点（像素） | 灰度值及分布 | 相关系数、差平方和、差绝对值 |
| 特征（点、线） | 特征向量 | 向量距离 |
| 面 | 面特征属性 | 距离、代价函数 |

- **相似性测度**：评价匹配实体相似程度的定量指标（由代价函数/目标函数度量）。
- **★三类匹配方法**：
  - **ABM（基于灰度 Area Based Matching）**：以灰度分布为基础，相似性测度如相关系数。
  - **FBM（基于特征 Feature Based Matching）**：以提取的特征为共轭实体、特征描述参数为匹配实体（SIFT 属此类）。
  - **RM（基于关系 Relationship Matching）**：以特征关系描述为匹配实体。
- **基于灰度匹配常用测度**：NCC（归一化互相关，相关系数越大相关性越强）、MAD、SAD（差绝对值和）、SSD（差平方和，S越小越匹配）、MSD、SSDA、SATD。
- **匹配策略/约束**（缩小搜索、提高可靠性）：金字塔影像、灰度相关约束、三角形几何约束、**核线几何约束**、视差梯度约束、**双向一致性约束**、铅垂线约束。
- **匹配过程**：将"目标区（模板）"相对"搜索区（>目标区）"不断移动，按匹配测度搜索最优匹配得同名点。
- **单基线 vs 多基线匹配**：
  - 单基线（双片匹配）：病态问题、无多余观测。
  - **多基线（多片匹配）**：减少信息盲区（解决遮挡）、有多余观测、提高可靠性、交会角小易匹配、多光束前方交会提高精度。

## ★本讲高频考点速记
1. SIFT = 尺度不变特征变换；4提取步 + 2匹配步。
2. 四大不变性：旋转、尺度、亮度、仿射。
3. 尺度空间=模糊程度；高斯核是唯一产生多尺度空间的核；DoG=相邻高斯影像相减≈LoG。
4. DoG 极值检测：与 **26 个邻点**比较。
5. 兴趣点定位：Taylor拟合子像素插值 + Hessian去边缘 + 去低对比度。
6. 主方向=直方图主峰，辅方向=**80%**能量峰。
7. **描述子 = 4×4×8 = 128 维**，欧氏距离匹配，最近邻/次近邻比值。
8. **RANSAC + DLT 求单应矩阵 H**（随机选4对点）消误匹配。
9. 匹配三方法：**ABM/FBM/RM**；灰度测度 NCC/SSD/SAD。
10. 多基线匹配优于单基线（解决遮挡、有多余观测）。
