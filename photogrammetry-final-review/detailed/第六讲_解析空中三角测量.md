# 第六讲 解析空中三角测量

> ★本讲承接双像解析，讲**如何用少量控制点加密出大量控制点**。六大板块：**概述 / 航带法 / 独立模型法 / 光束法 / GPS辅助 / POS辅助**。**三种平差方法的对比（平差单元、观测值、未知数、数学模型、精度）是必考大题**；精度分布规律、系统误差改正也是高频点。

---

## 〇、预备：坐标系转换（"一步之遥"）
双像解析求出的是**地面摄影测量坐标**，还需转成大地坐标；多模型还要依次连接。
- **摄影测量坐标系 → 地面摄影测量坐标系 → 大地坐标系**（三级转换）。
- 平面变换用二维相似变换（含 a=λcosθ, b=λsinθ）；**不同手系变换 |R|=−1，同手系 |R|=+1**。
- **地球曲率引起的高程差**：$\Delta Z_R = \dfrac{D^2}{2R} = \dfrac{X_{tp}^2 + Y_{tp}^2}{2R}$（R 为地球曲率半径），$Z_{tp} = \lambda Z_t + \Delta Z_R$。

## 一、解析空中三角测量概述

### 问题提出与基本概念
- 传统每个像对最少需实测 4 个控制点，外业量大。**能否只测少量外业控制点，内业加密出每个像对所需控制点？**
- **解析空中三角测量**：以像片上量测的像点坐标为依据，用较严密的数学模型、按最小二乘原理、用少量地面控制点为平差条件，用计算机解算测图所需地面控制点空间坐标。又称**电算加密**。
- **输入**：大量同名像点坐标（观测值）+ 少量地面控制点坐标（已知条件）。
- **产出**：大量加密点的地面三维坐标 + 每张影像的外方位元素。
- 加密方法：插网法、插点法（三角网控制点加密）。

### 摄影测量作业过程（4步）
①航空摄影获取航片 → ②外业调绘与控制联测 → ③**用少量控制点摄影测量加密，解求每张像片的外方位元素**（即解析空三，最复杂环节）→ ④像片定向及测图。

### 基本意义与目的
- 意义：不接触目标即可测定位置形状；大范围同时点位测定、省野外工作量；不受通视限制；**区域内部精度均匀，不受区域大小限制**。
- 目的：为4D产品提供定向控制点和影像定向参数；三四等大地测量；界址点坐标；单元模型大量地面点坐标；近景/卫星/倾斜摄影测量。

### ★基本分类（必记）
- **按数学模型**：**航带法、独立模型法、光束法**。
- **按平差范围**：单模型、单航带、区域网。

### 基本信息
- **摄影测量信息**：控制点、连接点的影像坐标。
  - **连接点/人工连接点**：相邻影像同名点，建立影像间联系。类型：人工/仪器转刺点、标志点、明显地物点、数字影像匹配转点。
  - **格鲁伯点（6点法）**：定向连接点标准配置。
- **非摄影测量信息**：将网纳入物方坐标系的基准信息（大地测量观测值、像片外方位元素、相对控制条件等）。

### ★像点坐标系统误差及改正（4项，必考）
引起误差因素：**① 底片变形 ② 物镜畸变 ③ 大气折光 ④ 地球曲率**。
- **底片变形**：框标在角隅用双线性变换、在中央用比例缩放改正。
- **物镜畸变差**：$\Delta x = x'(k_0 + k_1 r^2 + k_2 r^4),\ r$ 为以像主点为极点的向径（径向畸变）。
- **大气折光**：引起像点径向变形。
- **地球曲率**：$\Delta r = \dfrac{r^3 H}{2f^2 R}$，以**像底点 n** 为极点的向径。
- 总改正：$x = x' + \Delta x_{底} + \Delta x_{畸} + \Delta x_{折} + \Delta x_{曲}$。

## 二、航带法空中三角测量（原理，精度低）

### 基本思想
- 把许多立体像对的单个模型连成一个**航带模型**，视为单元模型处理，**消除航带模型中累积的系统误差**，整体纳入测图坐标系，确定加密点地面坐标。
- 单模型连成航带时，各模型的**偶然误差和残余系统误差会传递到下一个模型**（"传话游戏"式累积）。

### ★基本流程（6步）
1. 像点坐标量测及**系统误差改正**；
2. 立体像对**连续法相对定向**，建立单个模型；
3. 模型连接（**比例尺归化**），构建**自由航带网**；
4. 航带模型**绝对定向**（空间相似变换纳入地面摄测坐标系）；
5. 航带模型**非线性改正**（多项式拟合消除扭曲变形）；
6. 加密点坐标计算。

### 关键点
- 相对定向后各模型：**坐标轴向彼此平行，但比例尺各不相同、坐标原点也不一致**。
- **模型连接的实质 = 比例尺归化**：以相邻两模型重叠区**三个连接点高度相等**为条件，逐个归化比例尺、统一原点。连接系数 $k = (k_1+k_3+k_5)/3$（取三点均值提高精度）。
- **非线性改正**：用多项式（三次或二次）逼近变形曲面，最小二乘拟合，改正坐标。
- **航带法区域网平差**：以单航带为基础，把几条航带/整个测区作为整体同时求全部待定点坐标。

## 三、独立模型法空中三角测量（高精度）

### 基本思想
- 把**独立的单元模型视为刚体**，利用模型间公共点连接成区域。连接中每个单元模型只做**旋转、缩放、平移（三维相似变换7参数）**，使公共点坐标尽量一致、控制点摄测坐标与地面坐标尽量一致，改正数平方和最小，按最小二乘求待定点地面摄测坐标。

### 基本流程
1. 按单独法相对定向建立单元模型（含摄站点坐标）；
2. 各单元模型分别作三维线性变换（7参数），逐点法化组成总法方程；
3. 建立改化法方程，循环分块法求每个模型的 7 个参数；
4. 用 7 参数计算待定点坐标，公共点取均值。
- 数学模型：单个模型的空间相似变换 $\begin{bmatrix}X\\Y\\Z\end{bmatrix} = \lambda R\begin{bmatrix}X_g\\Y_g\\Z_g\end{bmatrix} + \begin{bmatrix}X_{tp}\\Y_{tp}\\Z_{tp}\end{bmatrix}$。

## 四、光束法空中三角测量（最严密，精度最高）★

### 基本思想
- 以**一个摄影光束（一张像片）**为平差基本单元，以**共线方程**为基础方程。通过各光束在空间旋转和平移，使公共点光线**最佳交会**，整个区域纳入控制点坐标系。在"公共点坐标应相等、控制点加密坐标应与地面摄测坐标一致、误差平方和最小"条件下，**同时解求各像片外方位元素和加密点地面坐标**。

### 基本流程
1. 确定像片外方位元素和地面点坐标**近似值**（常用航带法加密成果作初值——最佳初值）；
2. 逐步建立误差方程和改化法方程；
3. 解求改化法方程；
4. 求每张像片外方位元素；
5. **多片前方交会**求待定点地面坐标，公共连接点取均值。
- 误差方程：$V_{ij} = A_{ij}\Delta_j + B_{ij}\dot\Delta_i - l_{ij}$（每张像片外方位元素6未知数占6×6，每点3未知数占3×3）。
- 迭代求解：反复用地面点近似值求外方位元素、再求新坐标，直到收敛。

### 解析摄影测量定位全过程
航空摄影 → 外业控制（像控测量得GCP）→ **空三加密**（解算像片外方位元素）→ **前方交会**求地面点坐标。

### ★★三种方法对比（必考大题！背下这张表）
| 比较项 | 航带法 | 独立模型法 | 光束法 |
|---|---|---|---|
| **平差单元** | 航带 | 单元模型 | 单张像片（光束） |
| **观测值** | 各点概略地摄坐标 | 模型坐标 | **像点坐标** |
| **未知数** | 各航带非线性变形改正系数 | 各模型空间相似变换参数+加密点坐标 | 各像片外方位元素+加密点坐标 |
| **数学模型** | 多项式 | 空间相似变换公式 | **共线方程** |
| **原理** | 近似 | 严密 | **最严密** |
| **精度** | 低 | 高 | **最高** |
| **应用** | 小比例尺低精度加密 | 测图加密、低级大地三角网、地籍测界址点 | 高精度加密 |

## 精度评定与误差分布规律（★易考）
- **理论精度**：由改正数方差-协方差阵求得；单位权中误差 $m_0 = \sqrt{\dfrac{pvv}{n-t}}$，$m_i = m_0\sqrt{Q_{ii}}$。
- **实际精度**：用野外实测检查点作真值，$m_{xy}, m_z$ 与加密结果比较。
- **误差分布规律**：
  1. 三种方法**最弱点在区域网四周**，内部精度均匀 → **平面控制点应布设在区域四周**。
  2. 控制点稀疏时精度随区域增大而降低；**增大旁向重叠可提高平面坐标理论精度**。
  3. 周边密集布点时：航带法精度<一条航线测点精度；独立模型法≈一个单元模型测点精度；**光束法区域理论精度不随区域大小改变**。
  4. 高程理论精度取决于**控制点定向跨度**，与区域大小无关。

## 五、GPS辅助空中三角测量

### 基本思想与目的
- 用机载GPS + 地面基准站GPS接收机同步观测，配合快门开启脉冲，经**载波相位差分定位后处理**获取曝光时刻摄站三维坐标，作为**带权观测值**引入区域网平差。
- **目的**：极大减少甚至免除地面控制点，节省外业、缩短周期、降成本、提效率。

### 关键流程与要点
1. 航摄系统改造及**偏心分量测定**（GPS天线相位中心 ≠ 摄影投影中心，需测偏心矢量）；
2. 带GPS的航空摄影（0.5~1.0s更新率）；
3. **Kalman滤波**求各历元天线坐标，**内插**曝光时刻GPS摄站坐标；
4. GPS摄站坐标与摄影测量数据**联合平差**。
- 偏心关系：$\begin{bmatrix}X_A\\Y_A\\Z_A\end{bmatrix} = R\begin{bmatrix}X\\Y\\Z\end{bmatrix} + \begin{bmatrix}X_S\\Y_S\\Z_S\end{bmatrix}$。
- GPS坐标可能含**系统误差**（漂移），误差方程加入**线性漂移改正项** $a + (t-t_0)b$；需 **3~4 个地面控制点**做坐标转换和系统误差消除。
- 推荐地面控制方案：**4角平高控制点 + 2排高程控制点**，或 **4角平高 + 2条垂直构架航线**。
- 优点：外业只需少量控制点、内业全自动、精度均匀可靠（航摄费增约15%）。

## 六、POS辅助空中三角测量

### 基本思想
- **POS = GPS + IMU**（定位定姿系统）。安装在航摄平台上，获取曝光时刻摄站的**空间位置和姿态**（直接测定 6 个外方位元素 Xs,Ys,Zs,φ,ω,κ），作为观测值引入区域网平差。
- 两种模式：**直接传感器定向**（直接前方交会）、**集成传感器定向**（POS辅助空三，带权观测值参与平差，精度更高）。
- 相比GPS辅助（只提供位置），POS还提供**姿态**，可实现无控制点直接定向。

## ★本讲高频考点速记
1. 空三定义（电算加密）：少量控制点→加密大量控制点+外方位元素。
2. 分类：航带法/独立模型法/光束法；单模型/单航带/区域网。
3. 系统误差4项改正：底片变形、物镜畸变、大气折光、地球曲率。
4. 航带法6步流程；模型连接实质=比例尺归化；非线性改正=多项式拟合。
5. **三种方法对比表**（平差单元、观测值、未知数、数学模型、原理、精度）——最重要。
6. **光束法最严密、精度最高**，以共线方程为基础、外方位元素+加密点坐标同时解算。
7. 精度分布：控制点布在**四周**；光束法精度不随区域大小变化。
8. GPS辅助：差分定位得摄站坐标作带权观测值，需3~4控制点消系统误差；POS辅助=GPS+IMU直接测外方位元素。
