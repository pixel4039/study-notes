# 第二讲 摄影测量基本知识

> 两大板块：**① 航空摄影基本知识 ② 中心投影知识基础**。本讲概念极多，是**名词解释、填空、简答**的高发区，务必记牢各参数的定义、数值要求与几何意义。

---

## 引言：摄影测量的数学基础
- 数学工具：线性/非线性优化（最小二乘、梯度下降、高斯-牛顿、LM法）、统计（高斯分布、参数估计、RANSAC、蒙特卡洛）、多项式代数、线性代数与矩阵论、向量代数与空间解析几何、**射影几何**（齐次坐标、二维/三维射影变换）。
- 基本几何约束：相机投影模型、**双视图对极约束（本质矩阵E、基础矩阵F、单应矩阵H）**、三维重建（相似/仿射/射影重构）、多视图约束（三焦张量）。

## 一、航空摄影基本知识

### 1. 成像原理：小孔成像与透视
- 摄影测量成像原理 = **基于小孔成像的一点透视（中心投影）成像**。
- **灭点（Vanishing Point）**：透视投影中，**不平行**于投影面的一束平行线，其投影会聚于一点，即灭点；可看作**无限远处一点在投影面上的投影**。（平行于投影面的平行线投影仍平行。）
- **透视类型**（按灭点数）：
  - **一点透视（平行透视）**：1个灭点，能产生纵深感。
  - **两点透视（成角透视）**：2个灭点，最符合视觉习惯，富有立体感。
  - **三点透视**：3个灭点（俯视/仰视）。
- 虚拟像平面：小孔成像得倒像，不便观察计算，故设计一个**虚拟像平面（正片位）**作为观察和计算依据。

### 2. 航摄相机结构与关键参数（★名词解释高频）
- 三大部件：暗盒（Magazine）、机身（Body）、镜筒（Lens cone）。
- **航摄仪焦距**：物镜节点到焦点的距离。
- **像片主距 f**：物镜**后节点到像平面**的距离（成像用的关键值）。按主距分：
  - 长焦距 >200mm；中焦距 100~200mm；短焦距 <100mm。
- **像场**：物镜焦面上中央成像清晰的范围；**像场角**：像场直径对物镜后节点的夹角。
  - 常角 <75°；宽角 75°~100°；特宽角 >100°。
- **像幅**：小像幅18×18、正常像幅**23×23mm**、大像幅30×30mm。

### 3. 框标（★必考，两种类型+作用）
- **框标（Fiducial Marks）**：承片框四边**中点**或四**角隅**的标志，随地物同时构像于胶片上。
  - **机械框标**：像平面框架每边**中点**的标记。
  - **光学框标**：像平面框架每**角隅**的标记。
- **框标的作用**：
  1. 两两相对框标连线（正交或对角线）的**交点 = 承片框中心 = 像主点**，是**像点坐标系（框标坐标系）的原点**。
  2. 由相对框标连线交点确定**像主点**位置。
  3. 框标识别与定位是**数字影像内定向**、提取影像数据的关键步骤，其中心坐标精度直接影响成果精度。
- 数字影像没有框标 → 用**像素坐标系**定义坐标系与原点。

### 4. 成像几何
- **框幅式成像**：主光轴垂直像平面，一次曝光获取整幅（面阵）。有负片和正片。
- **CCD/线阵推扫成像**：焦平面上二维CCD阵列，图像是像素格网；卫星多用**推扫式**。

### 5. 航高（★相对/绝对定义 + 关系）
- **航高**：摄影瞬间摄影机相对某水准面的高度，向上为正。
  - **相对航高**：摄影机物镜相对某一**基准面**的高度。
  - **绝对航高**：相对**平均海平面**的真实海拔高度。
- 关系：$H_{绝} = H_{相对} + H_{地}$（基准面海拔）。

### 6. 航摄比例尺（★核心公式）
$$\frac{1}{m} = \frac{l}{L} = \frac{f}{H}$$
- $f$=主距，$H$=航高，$l$=像片上线段，$L$=对应地面长度。
- **像片倾斜、地形起伏时 m 不是常数**（航片无统一比例尺）。
- 取摄区**平均高程面**为基准面时，物镜中心到该面的距离称为**摄影航高**。
- **航摄比例尺与测图比例尺关系**（易考结论）：
  - 测**小比例尺**图：航摄比例尺**大于**测图比例尺。
  - 测**中比例尺**图：航摄比例尺**接近**测图比例尺。
  - 测**大比例尺**图：航摄比例尺**小于**测图比例尺。

### 7. 航摄过程关键概念（★名词解释密集区）
- **摄站点 S**：曝光瞬间摄影机物镜所在的空间位置。
- **摄影基线 B**：航向相邻两摄站间的距离。
- **像对**：航向相邻两张像片组成一个像对。
  - 理想像对：两像片水平、基线水平；正直像对：两像片水平、基线不水平；竖直像对：两像片不严格水平、基线不水平。
- **重叠度**（★必记数值）：
  - **航向重叠度** $p_x\% = \frac{p_x}{L_x}\times100\%$，要求 **60%~65%，不小于53%**。
  - **旁向重叠度** $p_y\% = \frac{p_y}{L_y}\times100\%$，要求 **30%~35%，不小于15%**。
  - 三度重叠：航向三张相邻像片的公共重叠（空三连接点、像片定向需要）。
  - 地面起伏大应增大重叠度；倾斜/大重叠摄影可 航向>80%、旁向40%~60%；**三线阵传感器有100%重叠度**。
- **像片倾角 α**：主光轴偏离铅垂线的夹角，要求 **<3°**（近似垂直摄影，<30°属摄影测量范畴）。
- **航线弯曲度**：航带两端像片主点连线距离 L 与偏离该直线最远像主点的垂距 δ 之比 = $\frac{\delta}{L}\times100\%$，**通常不大于3%**。
- **像片旋偏角 κ**：相邻两像片主点连线 与 沿航带方向的两框标连线间的夹角，要求 **<6°**（过大会减少立体像对有效范围）。

### 8. ★航空摄影飞行质量要求（汇总记忆表）
| 指标 | 要求 |
|---|---|
| 航片重叠度 | 航向>60%，旁向>30% |
| 像片倾斜角 | <3° |
| 航线弯曲度 | <3% |
| 像片旋偏角 | <6° |
| （航高差） | <5% |

## 二、中心投影知识基础

### 1. 投影分类
- **投影**：空间点按一定方式在平面上的构像（要素：物点、像点、投射线、像面）。
- **平行投影**（投射线互相平行）→ 特例：**正射投影（垂直投影）**，地形图属此类。
- **中心投影（Central Projection）**：所有投射线（或延长线）都通过一个**固定点（投影中心S）**。**航摄像片是地面的中心投影**。
- **正片位 / 负片位**：
  - 负片位：投影中心位于物与像**之间**（真实成像，倒像）。
  - 正片位：投影中心位于物与像**同侧**（虚拟像平面，正像，便于观察）。
- 航空摄影测量的主要任务之一：**将中心投影的航片转化为正射投影的地形图**。

### 2. 中心投影特征（"一般是…，存在特例"）
- 点的中心投影**一般是点**（特例：成像在无穷远）。
- 线段的中心投影**一般是线段**（特例：一个点/半直线/无穷远）。
- 相交线段的中心投影**一般是相交线段**（特例：变成一条直线）。
- 一组**不与承影面平行**的空间平行直线，中心投影为一**平面线束**，顶点称**合点**（这组平行线无穷远点的中心投影）。
- 平面曲线的中心投影一般是平面曲线；**空间曲线的中心投影是平面曲线**。

### 3. 透视变换与航摄像片特殊点线面（★难点，可能简答/画图）
- **透视变换**：两个平面间的中心投影变换。投影中心=透视中心，像点=透视，物点=投影。
- 基本要素：**S**摄影中心、**P**倾斜像片面、**T**地平面。
- **特殊面（3个）**：主垂面 W、真水平面 Es、遁面 R。
- **特殊线（6条）**：透视轴 tt、摄影方向线 VV、像片主纵线 vv、灭线 jj、真水平线 hihi、主垂线 SN。
- **特殊点（9个）**：像主点 o、地主点 O、像底点 n、地底点 N、主遁点 J、主迹点 v、主合点 i、像等角点 c、地等角点 C（另有主横线 hoho、等比线 hchc）。
- 三个关键特性：
  - **底点特性**：铅垂线在像平面的构像位于以**像底点 n** 为辐射中心的辐射线上。
  - **等角点特性**：由等角点 c、C 引出的透视对应线**无方向偏差**（方向角相等）。
  - **等比线特性**：等比线上构像比例尺 = 水平像片摄影比例尺，**不受像片倾斜影响**。

### 4. ★航摄像片与地形图的差异（必考简答）
1. **投影方式不同**：地形图=正射投影，航摄像片=中心投影。
2. **像点位移（两项误差）**：
   - **像片倾斜引起的像点位移**：以**等角点 c**为辐射中心；边缘误差>中心；**等比线上无此位移**；等角点辐射线上两点连线无方向偏差。
   - **地形起伏引起的像点位移**：由地面点相对基准面的**高差**引起；在以**像底点 n**为辐射中心的方向线上；边缘>中心。
   - （另有**物理因素引起的像点位移**：大气折光、物镜畸变差、地球曲率、底片变形。）
3. **比例尺不同**：地图有统一比例尺，航片无统一比例尺。
4. **表示方法不同**：地图为线划图，航片为影像图。
5. **内容表示不同**：地图需综合取舍。
6. **几何上不同**：航片可组成像对进行立体观察。

> 关键区分：**倾斜像点位移 → 以等角点c为中心；地形起伏像点位移 → 以像底点n为中心。** 这是最容易考的对比点！

## ★本讲高频考点速记
1. 框标：机械框标（边中点）/光学框标（角隅），交点=像主点=像坐标系原点。
2. 航摄比例尺 **1/m = f/H**；相对航高 vs 绝对航高，$H_绝=H_相对+H_地$。
3. 重叠度数值：航向60~65%(≥53%)、旁向30~35%(≥15%)；三度重叠。
4. 飞行质量：倾角<3°、旋偏角<6°、航线弯曲<3%。
5. 中心投影 vs 正射投影；正片位/负片位。
6. **两项像点位移**：倾斜→等角点c；地形起伏→像底点n。
7. 特殊点线面数量：面3、线6、点9。
